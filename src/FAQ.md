# 常见问题

以下列出的是人们经常问及的**关于本指南的**问题。这些**不是**关于学习日语的问题，而是关于 Yokubi 项目的问题。

### 你为什么盗用别人的作品？

Yokubi 几乎完全基于 [Sakubi](https://sakubi.neocities.org/) 这并非秘密。大部分课程、主题以及很多措辞都是直接复制、重新措辞和重新编排的，但指南的核心仍然是相同的。

然而，我**并没有**盗用 Sakubi。Sakubi 是一个开放项目，由 Sakubi 作者（我的一位好朋友）原样赠予社区。它以 CC0 许可协议发布，属于[公有领域](https://creativecommons.org/publicdomain/zero/1.0/)。除此之外，Sakubi 项目已被遗弃，自 2018 年以来没有收到任何更新。如果你仍然不相信我，我甚至与原作者进行了沟通，并获得了他在 Yokubi 上的祝福。

**Yokubi 是 Sakubi 的官方精神续作**

### 与 Sakubi 相比有哪些变化？

我最初试图记录与原指南相比的所有变化，无论大小。但我写得越多，想改的就越多，以至于现在的结构已经明显不同了。不过，我可以列出**一些**变化。这份列表并不完整，但应该能让你对投入的工作量有一个大致的了解。

#### 主要变化

- 课程已拆分为 mdbook 格式下的独立页面，而不是单一的独立网页。这使得浏览和与其他学习者分享单个课程更加容易。
- 在动词方面，放弃了"一段形式"和"五段形式"等说法，改用一段（ichidan）和五段（godan）术语
- 一些课程和语法点被拆分为单独的课程。Sakubi 有 54 课。Yokubi 有 63 课。
- 新增了关于否定状态的一课
- 新增了关于日语句子结构的第 0 课
- 对一些课程进行了重新排列和重新排序，使重要性优先级的排序更加合理
- 新增了关于 <span lang="ja">する</span> 动词的章节
- 新增了关于 <span lang="ja">ようとする</span> 和 <span lang="ja">てみる</span> 的一课（而非可选章节）
- 新增了关于 <span lang="ja">たり〜たり</span> 的专门一课
- 将一些"Part"合并到了一起。Sakubi 将第 1 节和第 2 节分成了多个部分。Yokubi 有 2 个节，每节有 2 个部分。
- 移除了"插曲"和"隐藏/可选课程"的概念。正在计划改为附录索引（尚未完成）
- 几乎所有的例句都从原生媒体重新选取，并按难度/适合性重新排序。除了少数例外，例句不会展示尚未涉及到的语法点。

#### 次要变化

- 改变了一些语气和口吻。很多第一人称的写作方式被改为更具吸引力的格式。
- 重新措辞了很多语法解释，减少行话的使用
- 更改了一些视频和资源的链接，替换为更好的（更新的）替代品
- 移除了很多有问题的陈述，这些陈述要么不正确，要么非常主观/难以验证其正确性
- 修正了大量拼写错误和直接错误

### 谁拥有 Yokubi？

Yokubi 作为一个项目由 [Morg](https://morg.systems) 发起，该项目在 [GitHub](https://github.com/Morgawr/yokubi) 上公开维护。然而，项目本身是一个社区项目，对所有人的讨论、反馈和贡献开放。很多人帮助提供了想法、修正、建议、校对、测试等。如果你也想加入他们，你可以在 GitHub 上提交 issue，或者直接加入我们的 [Discord 服务器](https://discord.gg/KZj4dVFDzu)。

### 是什么让 Yokubi 优于其他语法指南？

我对于本指南应该是什么样子有着非常明确的看法。如果你想学习日语，还有很多其他优秀的语法指南（和教科书）。Yokubi（和 Sakubi）并不是"最好的"或"终极"指南之类的。它们只是一种处理问题的方式。正如其前身一样，Yokubi 侧重于（通常是）简短、直截了当且非常直接的解释。它**不会**手把手全程牵着你走。它**期望**读者付出一些努力和主动的投入。去查询事物，实现独立自主的学习。它也**期望**学习者更多地关注沉浸和自然内容，而不是纠结于最细微的语法解释细节。

如果你认为这符合你对日语学习的想法，那么请从[准备开始](./Before-you-begin.md)页面开始，着手学习。如果不适合，市面上还有其他的指南。**Yokubi 并不期望适用于所有人**。

### Yokubi 是什么意思？

正如 [Imabi - <span lang="ja">今日</span>](https://imabi.org/) 和 [Sakubi - <span lang="ja">昨日</span>](https://sakubi.neocities.org/) 是在 <span lang="ja">今日</span>（今天）和 <span lang="ja">昨日</span>（昨天）这两个词上的双关语一样，Yokubi 是在 <span lang="ja">翌日</span>（次日）这个词上的双关语。这也是我和朋友们之间的一个内部笑话，其中 "yoku"（<span lang="ja">よく</span>）被**错误地**用作 <span lang="ja">よくある文法</span> 或 <span lang="ja">よくある言葉</span>（="常见语法"或"常见词语"）的简写形式。

就这样，Yokubi 就是**通用**语法指南。
