# 基本礼貌与 <span lang="ja">ます</span> 形

现在让我们来谈谈礼貌。在日语中，礼貌的语气表现为一种特定的语法特征。这意味着当你有礼貌地说话时，你造句、活用，甚至你选择的词都会与随意说话时不同。有不同程度的礼貌，我们这里只涵盖基本的内容。

基本的礼貌动词是通过将助动词 <span lang="ja">ます</span> 附在词干上来构成的（这个词干通常被称为"<span lang="ja">ます</span>-词干"）。对于一段动词，<span lang="ja">ます</span> 附着在动词去掉 <span lang="ja">る</span> 的部分上。对于五段动词，将 u 变为 i，然后附上 <span lang="ja">ます</span>。

| 随意     | 礼貌       |         |
|------------|--------------|---------|
| <span lang="ja">食べ</span>**<span lang="ja">る</span>** | <span lang="ja">食べ</span>**<span lang="ja">ます</span>** | To eat  |
| <span lang="ja">行</span>**<span lang="ja">く</span>**   | <span lang="ja">行</span>**<span lang="ja">きます</span>** | To go   |
| <span lang="ja">切</span>**<span lang="ja">る</span>**   | <span lang="ja">切</span>**<span lang="ja">ります</span>** | To cut  |
| <span lang="ja">歌</span>**<span lang="ja">う</span>**   | <span lang="ja">歌</span>**<span lang="ja">います</span>** | To sing |

<div class="warning">
不要在 <span lang="ja">です</span> 后面附加 <span lang="ja">ます</span>，这行不通。虽然你可能会在一些媒体中看到故意说错误日语的角色这样使用。不要模仿他们。
</div>

在[第9课](./Lesson9.md)中，我们提到五段动词的过去时来自"i"词干。虽然对于某些动词（<span lang="ja">死ぬ</span> \-\> <span lang="ja">死んだ</span>），这个词干在过去形中显示了一个例外，但 <span lang="ja">ます</span> 不会发生这种情况。

以下是每种五段动词的过去时和 <span lang="ja">ます</span> 形之间的比较。不要记忆这个。

| 随意     | 礼貌       | 过去         |
|------------|--------------|--------------|
| <span lang="ja">切</span>**<span lang="ja">る</span>**   | <span lang="ja">切</span>**<span lang="ja">ります</span>** | <span lang="ja">切</span>**<span lang="ja">った</span>**   |
| <span lang="ja">殺</span>**<span lang="ja">す</span>**   | <span lang="ja">殺</span>**<span lang="ja">します</span>** | <span lang="ja">殺</span>**<span lang="ja">した</span>**   |
| <span lang="ja">死</span>**<span lang="ja">ぬ</span>**   | <span lang="ja">死</span>**<span lang="ja">にます</span>** | <span lang="ja">死</span>**<span lang="ja">んだ</span>**   |
| <span lang="ja">選</span>**<span lang="ja">ぶ</span>**   | <span lang="ja">選</span>**<span lang="ja">びます</span>** | <span lang="ja">選</span>**<span lang="ja">んだ</span>**   |
| <span lang="ja">読</span>**<span lang="ja">む</span>**   | <span lang="ja">読</span>**<span lang="ja">みます</span>** | <span lang="ja">読</span>**<span lang="ja">んだ</span>**   |
| <span lang="ja">買</span>**<span lang="ja">う</span>**   | <span lang="ja">買</span>**<span lang="ja">います</span>** | <span lang="ja">買</span>**<span lang="ja">った</span>**   |
| <span lang="ja">動</span>**<span lang="ja">く</span>**   | <span lang="ja">動</span>**<span lang="ja">きます</span>** | <span lang="ja">動</span>**<span lang="ja">いた</span>**   |
| <span lang="ja">急</span>**<span lang="ja">ぐ</span>**   | <span lang="ja">急</span>**<span lang="ja">ぎます</span>** | <span lang="ja">急</span>**<span lang="ja">いだ</span>**   |
| <span lang="ja">持</span>**<span lang="ja">つ</span>**   | <span lang="ja">持</span>**<span lang="ja">ちます</span>** | <span lang="ja">持</span>**<span lang="ja">った</span>**   |

由于 <span lang="ja">ます</span> 本身是一个像其他动词一样结尾的助动词，它也可以进一步被活用以提供时态、否定等。但请注意，否定形的活用方式是不规则的。

|                  |             |
|------------------|-------------|
| <span lang="ja">死にます</span>         | to die      |
| <span lang="ja">死にました</span>       | died        |
| <span lang="ja">死にません</span>       | to not die  |
| <span lang="ja">死にませんでした</span> | did not die |

<span lang="ja">ません</span> 本身没有自己的过去时。要使用 <span lang="ja">ません</span> 的过去时，你需要附上 <span lang="ja">でした</span>，即 <span lang="ja">です</span> 的过去形。

在现代日语中，还有另一种构成动词礼貌否定过去形的方式，就是直接在非礼貌的否定过去形后面加上 <span lang="ja">です</span>：

| 随意       | 礼貌           |
|--------------|------------------|
| <span lang="ja">死ななかった</span> | <span lang="ja">死ななかったです</span> |
| Did not die  | Did not die      |

这种使用 <span lang="ja">です</span> 而不是 <span lang="ja">ます</span> 的用法可以被视为稍微不那么礼貌，但已经变得越来越普遍，以至于完全正常使用，而且两者之间的区别非常细微，无法解释。你可以认为它们是一样的。

同样的情况也适用于 <span lang="ja">ないです</span> vs <span lang="ja">ません</span>，如 <span lang="ja">食べないです</span> vs <span lang="ja">食べません</span>。

最后，我们来看 <span lang="ja">っす</span>，这是 <span lang="ja">です</span> 或 <span lang="ja">ます</span> 的一种模糊发音版本，用于随意口语中。它比 <span lang="ja">です</span> 更不正式，但起相同的语法作用。它通常在你想显得随意但仍需遵循 <span lang="ja">です</span>/<span lang="ja">ます</span> 的社会期望的语境中使用。有点像穿着商务休闲装去上班。

<pre>
<span lang="ja">いいです</span> -> <span lang="ja">いい</span><b><span lang="ja">っす</span></b>
<span lang="ja">犬ですか</span> -> <span lang="ja">犬</span><b><span lang="ja">っす</span></b><span lang="ja">か</span>
</pre>

<span lang="ja">っす</span> 有时也用来缩约 <span lang="ja">ございます</span>，这是 <span lang="ja">ある</span> 的敬语版本。再次强调，去掉了正式性，但保留了礼貌。

<span lang="ja">おはようございます</span> \-\> <span lang="ja">おはようっす</span> \-\> <span lang="ja">おっす</span> （是的，真的）
