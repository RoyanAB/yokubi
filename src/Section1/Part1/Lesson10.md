# <span lang="ja">て</span> 形

如果这节课看起来很难，不用担心。我们只是在讲基础知识，你还不被期望现在就内化这些概念。

有一种以 <span lang="ja">て</span> 结尾的动词形式。这种动词形式主要有两种用法：

- 将动作或语句连接在一起；
- 为动作或语句提供额外的细微含义。

<span lang="ja">て</span> 形是通过将过去形的 <span lang="ja">た</span> 替换为 <span lang="ja">て</span> 来构成的。请记住，这不是某种次要的过去时。<span lang="ja">て</span> 形在其形式中不编码任何"时态"概念。它不告诉我们句子是过去还是现在。

<pre>
<span lang="ja">殺</span><b><span lang="ja">して</span></b><span lang="ja">　切</span><b><span lang="ja">って</span></b><span lang="ja">　買</span><b><span lang="ja">って</span></b><span lang="ja">　死</span><b><span lang="ja">んで</span></b><span lang="ja">　食</span><b><span lang="ja">べて</span></b>
</pre>

<div class="warning">
五段动词 <span lang="ja">行く</span> 是一个例外，它的 <span lang="ja">て</span> 形是 <span lang="ja">行って</span>，而<b>不是</b> <span lang="ja">行いて</span>。
</div>

形容词也可以有 <span lang="ja">て</span> 形。这是通过将 <span lang="ja">い</span> 变为 <span lang="ja">くて</span> 来实现的。

|               |                |
|---------------|----------------|
| <span lang="ja">赤</span><b><span lang="ja">い</span></b>   | <span lang="ja">赤</span><b><span lang="ja">くて</span></b>  |
| <span lang="ja">怖</span><b><span lang="ja">い</span></b>   | <span lang="ja">怖</span><b><span lang="ja">くて</span></b>  |
| <span lang="ja">面白</span><b><span lang="ja">い</span></b> | <span lang="ja">面白</span><b><span lang="ja">くて</span></b>|
| <span lang="ja">い</span><b><span lang="ja">い</span></b>   | <span lang="ja">よ</span><b><span lang="ja">くて</span></b>  |

<div class="warning">
形容词 <span lang="ja">いい</span> 是一个例外，它变为 <span lang="ja">よくて</span>。
</div>

这也适用于否定活用，这也是我们如何将 <span lang="ja">て</span> 形应用于否定动词的方式。

|          |            |
|----------|------------|
| <span lang="ja">赤くな</span><b><span lang="ja">い</span></b> | <span lang="ja">赤くな</span><b><span lang="ja">くて</span></b> |
| <span lang="ja">食べな</span><b><span lang="ja">い</span></b> | <span lang="ja">食べな</span><b><span lang="ja">くて</span></b> |

我们见过"A 是 B"和"A 做 B"类型的句子。随着 <span lang="ja">て</span> 形的引入，我们将看到第一个"A 做 B，然后 D 做 C"类型句子的例子。在最基本的句法层面上，A<span lang="ja">て</span>B 表示"A 发生，然后 B"。它可以标记时间顺序，或因果关系。它允许我们造出有多个按顺序排列的动词的更长的句子。注意，两边的主题不必相同。

<pre>
<span lang="ja">察しがよく</span><b><span lang="ja">て</span></b><span lang="ja">助かる</span>
你很有悟性，帮了大忙。（直译：你悟性好，这帮助了我）
</pre>

注意：在 <span lang="ja">て</span> 后面是否使用逗号不重要。一般来说，逗号在日语中没有语法作用，但它们可以让解析变得更容易。

语句 <span lang="ja">察しがよくて</span> 后面跟着语句 <span lang="ja">助かる</span>。<span lang="ja">助かる</span> 不是修饰 <span lang="ja">よくて</span>。相反，第一个语句为第二个语句设置上下文。在日语中，先构建一个初始语句，接着是一个 <span lang="ja">て</span> 活用，然后是一个描述你对第一个语句的主观感受或情感的附加语句，这是很常见的。你感到欣慰（<span lang="ja">助かる</span>），因为对方具有 <span lang="ja">察しがいい</span> 的品质。

<pre>
<span lang="ja">遅くなっ</span><b><span lang="ja">て</span></b><span lang="ja">すみません</span>
对不起我迟到了（直译："我迟到了，很抱歉"）
</pre>

在常见的搭配 <span lang="ja">〜てすみません</span> 中，<span lang="ja">て</span> 之前的部分表示道歉的原因。

<pre>
<span lang="ja">彼女の目を見</span><b><span lang="ja">て</span></b><span lang="ja">、頷いた</span>  
我看着她的眼睛<b>然后</b>点了点头
</pre>

在这个例子中，<span lang="ja">て</span> 只是将语句作为简单的动作序列串联起来。如前所述，<span lang="ja">て</span> 形不提供任何过去或现在的时态概念。时态是由序列中最后一个动词传达的。<span lang="ja">頷いた</span> 是过去形，所以我们可以假设整个语句是过去时。

有时 <span lang="ja">て</span> 形让动词充当主题。这样的话，它可以看起来像一个名词。它真的不是一个名词，只是在这种情况下看起来像。在这种用法中，它后面常常跟着 <span lang="ja">は</span> 助词。

<pre>
<span lang="ja">腹が減っ</span><b><span lang="ja">ては</span></b><span lang="ja">戦はできない</span>
肚子饿了就无法打仗。
</pre>

<span lang="ja">ない</span> 的 <span lang="ja">て</span> 形，即 <span lang="ja">なくて</span>，通常带有"没有做 X，所以 Y"的细微含义。但这只是一种细微含义。这仅仅是它可能的用法之一。

<pre>
<span lang="ja">早く起き</span><b><span lang="ja">なくて</span></b><span lang="ja">仕事に遅れた</span>  
没有早起，结果上班迟到了。（原因）
</pre>
