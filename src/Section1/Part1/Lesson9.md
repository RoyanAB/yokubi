# 动词的过去形

通过在特定的"词干"上附加 <span lang="ja">た</span> 或 <span lang="ja">だ</span> 来构成动词的过去时。

词干对于五段动词来说很重要，因为五段动词有很多词干，而且经常呈现不规则性。

90% 的情况下，你只使用三种词干。以下是一个例子：

| 普通形 | "a" 形 | "i" 形 |
|------------|----------|----------|
| <span lang="ja">殺す</span>       | <span lang="ja">殺さ</span>     | <span lang="ja">殺し</span>     |
| <span lang="ja">死ぬ</span>       | <span lang="ja">死な</span>     | <span lang="ja">死に</span>/<span lang="ja">死ん</span>|
| <span lang="ja">書く</span>       | <span lang="ja">書か</span>     | <span lang="ja">書き</span>/<span lang="ja">書い</span>|

注意："a" 形和 "i" 形不是真正的术语，这只是例子来说明这些词干结尾的元音。我们现在不需要知道这些形式的日语语言学名称，但如果想查的话可以去查。

以下是这些词干的用法示例：

<pre>
<span lang="ja">殺</span><b><span lang="ja">す</span></b>
他杀

<span lang="ja">殺</span><b><span lang="ja">さ</span></b><span lang="ja">ない</span>
他不杀

<span lang="ja">殺</span><b><span lang="ja">し</span></b><span lang="ja">た</span>
他杀了
</pre>

没错。像时态和否定这样的基本内容只使用三种词干。还有更多词干，但它们出现的频率较低。我们稍后会讨论它们。

有了这些准备，我们可以正式介绍过去时了。我们已经看到了 <span lang="ja">だ</span> 和 <span lang="ja">です</span> 的过去形式，即 <span lang="ja">だった</span> 和 <span lang="ja">でした</span>。我们在上面的词干例子中也看到了 <span lang="ja">殺した</span>。

你猜对了 <span lang="ja">た</span> 是使其成为过去时的部分。困难的部分在于，五段动词过去时的词干不像否定形的词干那样规则。它取决于普通形原始音节结尾。以下是所有可能性：

<pre>
<span lang="ja">切</span><b><span lang="ja">る</span></b>	<span lang="ja">殺</span><b><span lang="ja">す</span></b>	<span lang="ja">死</span><b><span lang="ja">ぬ</span></b>	<span lang="ja">選</span><b><span lang="ja">ぶ</span></b>	<span lang="ja">読</span><b><span lang="ja">む</span></b>	<span lang="ja">買</span><b><span lang="ja">う</span></b>	<span lang="ja">動</span><b><span lang="ja">く</span></b>	<span lang="ja">急</span><b><span lang="ja">ぐ</span></b>	<span lang="ja">持</span><b><span lang="ja">つ</span></b>
<span lang="ja">切</span><b><span lang="ja">った</span></b>	<span lang="ja">殺</span><b><span lang="ja">した</span></b>	<span lang="ja">死</span><b><span lang="ja">んだ</span></b>	<span lang="ja">選</span><b><span lang="ja">んだ</span></b>	<span lang="ja">読</span><b><span lang="ja">んだ</span></b>	<span lang="ja">買</span><b><span lang="ja">った</span></b>	<span lang="ja">動</span><b><span lang="ja">いた</span></b>	<span lang="ja">急</span><b><span lang="ja">いだ</span></b>	<span lang="ja">持</span><b><span lang="ja">った</span></b>
</pre>

不要记忆这个表。有一些基于动词属于哪个假名行的规则：

- <span lang="ja">う</span>、<span lang="ja">つ</span> 或 <span lang="ja">る</span> 变成 <span lang="ja">った</span>
- <span lang="ja">す</span> 变成 <span lang="ja">した</span>
- <span lang="ja">ぶ</span>、<span lang="ja">む</span> 或 <span lang="ja">ぬ</span> 变成 <span lang="ja">んだ</span>
- <span lang="ja">く</span> 和 <span lang="ja">ぐ</span> 分别变成 <span lang="ja">いた</span> 和 <span lang="ja">いだ</span>

还有一些动词有有趣的不规则活用，例如 <span lang="ja">問う</span> 变成 <span lang="ja">問うた</span>。只有少数几个动词表现得像这样（<span lang="ja">乞う</span> \-\> <span lang="ja">乞うた</span> 是另一个）。你不需要记忆这些，只需知道它们存在即可，它们并不常见。

日语母语者不会在每次活用某个东西时都经历这个心理过程。这个过程对他们来说是第二天性。至少，如果你做一点日语阅读和听力，你就能一眼认出过去形。接触比记忆更强大。

一段动词的过去时很简单。只需将 <span lang="ja">る</span> 替换为 <span lang="ja">た</span>。

| 非过去 | 过去        |
|----------|-------------|
|<span lang="ja">見</span>**<span lang="ja">る</span>**  | <span lang="ja">見</span>**<span lang="ja">た</span>**    |
|<span lang="ja">食べ</span>**<span lang="ja">る</span>** | <span lang="ja">食べ</span>**<span lang="ja">た</span>** |

我们将在第12课中学习这个词干的另一种用法。

动词活用的规则比这更复杂，但一步一步来。
