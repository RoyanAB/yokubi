# <span lang="ja">で</span> 助词、<span lang="ja">では</span> 和 <span lang="ja">じゃ</span>

通常情况下，<span lang="ja">で</span> 标记动作背后的"情况"或"手段"，或动作发生的"地点"。

<pre>
<span lang="ja">刀</span><b><span lang="ja">で</span></b><span lang="ja">殺した</span>
He killed <b>with</b> a katana

<span lang="ja">バス</span><b><span lang="ja">で</span></b><span lang="ja">来た</span>
I came <b>by</b> bus

<span lang="ja">学校</span><b><span lang="ja">で</span></b><span lang="ja">働く</span>
To work <b>in</b> a school

<span lang="ja">金属</span><b><span lang="ja">で</span></b><span lang="ja">出来ている</span>
It is made <b>of</b> metal
</pre>

<span lang="ja">で</span> 最初来源于复合助词 <span lang="ja">にて</span> 的缩约。你可能仍然会在老式或正式的写作或公告中看到 <span lang="ja">にて</span>。

<span lang="ja">で</span> 可以充当 <span lang="ja">だ</span> 的 <span lang="ja">て</span> 形，作用于名词，就像 <span lang="ja">て</span> 形作用于动词一样。在这种用法中，它承担了与动词或形容词的 <span lang="ja">て</span> 形相同的典型连接用法。

<pre>
<span lang="ja">中学生</span><b><span lang="ja">で</span></b><span lang="ja">一人暮らしはできない</span>
To be a middle schooler <b>and</b> live alone is not possible
</pre>

有时区分格标记 <span lang="ja">で</span> 助词（位置、手段等）和 <span lang="ja">だ</span> 的 <span lang="ja">て</span> 形连接用法并不总是很清楚。然而，重要的是句子的含义仍然应该是清晰的。你不应该担心要为句子中使用的每个助词精确定位字典定义。如果你理解了意思，那就足够了。

这引出了复合助词 <span lang="ja">では</span>。这里的 <span lang="ja">は</span> 可以被视为主题标记，但 <span lang="ja">では</span> 组合在一起的含义不仅仅是其组成部分的总和。<span lang="ja">では</span> 的这种特定用法强调 <span lang="ja">では</span> 后面的陈述适用于由 <span lang="ja">では</span> 标记的名词的上下文和情况。

<pre>
<span lang="ja">公園</span><b><span lang="ja">では</span></b><span lang="ja">アイスが売られる</span>
Ice cream is sold <b>at</b> the park.
</pre>

这也延伸到状态表达，正如我们在[第1课](./Lesson1.md)中已经看到的。

<pre>
<span lang="ja">私は学生</span><b><span lang="ja">では</span></b><span lang="ja">ない</span>
I am not a student.
</pre>

作为提醒，<span lang="ja">では</span> 通常缩约为 <span lang="ja">じゃ</span>。一些方言可能使用与 <span lang="ja">だ</span> 不同的系词。例如 <span lang="ja">や</span>（常用于关西方言）和 <span lang="ja">じゃ</span>（在虚构媒体中常由老人使用以听起来老派）。这个 <span lang="ja">じゃ</span> 是不同于 <span lang="ja">では</span> 的其他东西的缩约。通常的 <span lang="ja">では</span> 不是系词。

<pre>
<span lang="ja">ペンじゃ</span>
'tis a pen. (it's a pen.)
</pre>
