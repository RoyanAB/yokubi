# 不规则动词和 <span lang="ja">する</span> 动词

现在我们已经涵盖了最常见的动词词干，我们可以讨论不规则动词了。日语只有两个主要的不规则动词，但它们非常常见。它们的活用完全是不规则的。注意，一些其他动词中还有一些其他的例外和特殊情况（我们已经见过 <span lang="ja">問うた</span>），但这两个是完全不同的，存在于它们自己的一个类别中。

<pre>
<span lang="ja">する</span>
做

<span lang="ja">来る</span>
来
</pre>

它们的活用如下：

<pre>
<span lang="ja">する</span>		做
<span lang="ja">した</span>		做了
<span lang="ja">しない</span>		不做
<span lang="ja">せよ</span> 或 <span lang="ja">しろ</span>	做！

<span lang="ja">くる</span>		来
<span lang="ja">きた</span>		来了
<span lang="ja">こない</span>		不来
<span lang="ja">こい</span>		来！
</pre>

有了 <span lang="ja">する</span>，我们可以引入一类新的动词：<span lang="ja">する</span> 动词。在日语中，你可以通过在大量名词后面加上 <span lang="ja">する</span> 来将它们变成动词。这不是所有名词都允许的，但很大一部分词会在词典中同时显示为"<span lang="ja">する</span> 动词"。这在源自中文的双汉字词中尤其常见。

<pre>
<span lang="ja">料理</span>
菜肴

<span lang="ja">料理</span><b><span lang="ja">する</span></b> 
做饭（直译："做菜肴"）
</pre>

它们的意思大多与它们的非动词结构相同，但在语法上，该名词实际上变成了一个真正的动词。

考虑以下句子：

<pre>
<span lang="ja">日本語を</span><b><span lang="ja">勉強する</span></b>
学习日语

<span lang="ja">日本語の勉強を</span><b><span lang="ja">する</span></b>
学习日语（直译："做日语的学习"）

*<span lang="ja">日本語を勉強をする</span>  
（不合语法的废话）
</pre>

第一个将 <span lang="ja">勉強</span> 当作动词，<span lang="ja">日本語を</span> 当作它的宾语。第二个将 <span lang="ja">勉強</span> 视为一个名词（由 <span lang="ja">日本語</span> 修饰），它是独立动词 <span lang="ja">する</span> 的宾语。第三个句子不合语法，因为它试图强行使用两个不关联的宾语，根本行不通。

虽然我们可以用两种略有不同的方式翻译第一个和第二个句子，以强调 <span lang="ja">勉強する</span> 和 <span lang="ja">勉強をする</span> 之间的区别，但实际上意思几乎是一样的。
