# 用关系从句获得细节

<div class="warning">
在语法中，"从句"是包含一个独立语句的短语。"已经结束了"是一个从句。"对不起，我迟到了"中的"我迟到了"是一个从句。像"我迟到了，因为堵车严重"这样的复合句包含多个从句。
</div>

关系从句是嵌入在其他语句中的语句。如果我们采用像"A 是 B"这样的结构，关系从句允许我们随心所欲地将其扩展为"A 是（是 C 的 B）"，然后是"A 是（是做 D 的 C 的 B）"。在英语中，这可能看起来很笨重，但在日语中这是非常普遍的。

在本课中，我们将讨论修饰名词的关系从句。修饰名词的关系从句就像"I used to live in"这样的东西，在句子"That is the house <ins>I used to live in</ins>"中。

要让一个动词直接修饰一个名词，你只需将动词紧贴在名词之前。这意味着日语动词基本上可以像形容词一样行事。

<pre>
<span lang="ja">食べる人</span>
吃的人

<span lang="ja">私にくれる人</span>
给我东西的人
</pre>

<span lang="ja">だ</span> 的基本形不能构成关系从句，因为它不能直接连接到名词的左边，但它的过去形可以。

<pre>
*<span lang="ja">必要だ食べ物です</span>
（不合语法）

<span lang="ja">好き</span><b><span lang="ja">だった</span></b><span lang="ja">食べ物</span>
我喜欢的食物
</pre>

在极少数情况下，这条规则可以被打破，但它是一条真正的规则，不是假的。不要造以 <span lang="ja">だ</span> 结尾的关系从句。

在较短的关系从句中，<span lang="ja">の</span> 可以用来代替用来标记主语的 <span lang="ja">が</span> 助词。这个 <span lang="ja">の</span> 作为 <span lang="ja">が</span> 的替代品**只能**出现在关系从句中，只要该从句中没有使用 <span lang="ja">を</span> 助词。

<pre>
<span lang="ja">君</span><b><span lang="ja">の</span></b><span lang="ja">来た場所</span>
你来的地方

<span lang="ja">私</span><b><span lang="ja">の</span></b><span lang="ja">食べたラーメン</span>
我吃过的拉面

<span lang="ja">私がラーメンを食べたレストラン</span>
我吃拉面的那家餐厅
</pre>

在最后一个例子中，<span lang="ja">の</span> 不能替换 <span lang="ja">が</span>，因为 <span lang="ja">私のラーメンを食べたレストラン</span> 会假定一个非常不同的、近乎荒谬的含义（"吃了我拉面的餐厅"）。
