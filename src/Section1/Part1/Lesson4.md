# 动词

日语有规则动词和不规则动词（<span lang="ja">動詞</span>）。然而，与英语及其大量不规则的动词活用（"read" \-\> "read"、"speak" \-\> "spoke" 等）不同，大多数日语动词是规则的，并遵循一套非常严格的模式规则。

日语有两大类规则动词：一段动词（<span lang="ja">一段</span>）和五段动词（<span lang="ja">五段</span>），它们的活用方式不同。一段的意思是"一行"，五段的意思是"五行"。这个术语对于理解它们的活用方式很重要。"行"指的是假名音节表（<span lang="ja">五十音図</span>），但你不需要记忆这个。

这里有一个例子，使用了 <span lang="ja">ます</span>（礼貌）形，我们将在[第17课](./Lesson17.md)中正式学习。不用费心去记忆这个表。

<pre>
<span lang="ja">見</span><b><span lang="ja">る</span></b><span lang="ja">／見ます</span>，一段动词。
<span lang="ja">切</span><b><span lang="ja">る</span></b><span lang="ja">／切</span><b><span lang="ja">り</span></b><span lang="ja">ます</span>，五段动词，以 <span lang="ja">る</span> 结尾。
<span lang="ja">買</span><b><span lang="ja">う</span></b><span lang="ja">／買</span><b><span lang="ja">い</span></b><span lang="ja">ます</span>，五段动词，以 <span lang="ja">う</span> 结尾。
<span lang="ja">死</span><b><span lang="ja">ぬ</span></b><span lang="ja">／死</span><b><span lang="ja">に</span></b><span lang="ja">ます</span>，五段动词，以 <span lang="ja">ぬ</span> 结尾。
</pre>

一段动词总是以 <span lang="ja">る</span> 结尾，在活用时只需去掉 <span lang="ja">る</span>。<span lang="ja">る</span> 前面的部分是动词的基本身份，不会改变。你可以称其为动词的"词干"。就活用而言，动词本身只有"一种形式"。

五段动词可以以多个音节中的一个结尾，不仅仅是 <span lang="ja">る</span>。在词典形中，这个音节总是以 u 音结尾，而不同的活用可以将其变为其他四个元音中的任何一个。这意味着它的末尾可以有五个不同元音中的任意一个。换句话说，"五种形式"或词干。

词典形中的五段动词可以以几个不同音节中的一个结尾，但它总是以（罗马字）元音"u"结尾。以下示例动词都是五段动词。

<pre>
<span lang="ja">切る</span>	<span lang="ja">殺す</span>	<span lang="ja">死ぬ</span>	<span lang="ja">選ぶ</span>	<span lang="ja">読む</span>	<span lang="ja">買う</span>	<span lang="ja">動く</span>	<span lang="ja">急ぐ</span>	<span lang="ja">持つ</span>
</pre>

一段动词总是以 <span lang="ja">る</span> 结尾

<pre>
<span lang="ja">食べる</span>	<span lang="ja">負ける</span>	<span lang="ja">変える</span>
</pre>

日语动词活用倾向于将词堆叠在一起，你最终会看到像 <span lang="ja">見られたくなかった</span> 这样长长的动词串。在未来的课程中，我们将开始学习这些长串活用背后的逻辑。随着你越来越多地接触这门语言，分解它们会越来越容易，所以不用担心。

本课只是建立理解后续课程所需的基础知识。
