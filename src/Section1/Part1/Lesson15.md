# <span lang="ja">な</span>-形容词

现在让我们看看 <span lang="ja">な</span>-形容词，以其后面跟着的 <span lang="ja">な</span> 命名。<span lang="ja">な</span>-形容词是看起来像名词的简单词，但它们可以通过 <span lang="ja">な</span> "助词"连接来定语修饰其他名词。

<pre>
<span lang="ja">バカ</span><b><span lang="ja">な</span></b><span lang="ja">子供</span>
傻孩子

<span lang="ja">大好き</span><b><span lang="ja">な</span></b><span lang="ja">人</span>
心爱的人
</pre>

现在我们学习了什么是关系从句，了解"定语地"这个词基本上就是"作为关系从句"的意思，这很好。

在句末，<span lang="ja">な</span>-形容词可以看起来像名词，就像名词一样，它们可以用 <span lang="ja">だ</span> 来结束句子。

<pre>
<span lang="ja">この部屋は静か</span><b><span lang="ja">だ</span></b>
这个房间很安静。
</pre>

<div class="warning">
提醒：在正常的口语日语中，以名词或 <span lang="ja">な</span>-形容词结尾的句子不需要以 <span lang="ja">だ</span> 结尾，尽管这是学校传统上教的内容。<span lang="ja">この部屋は静か</span> 已经是一个完整、独立且有效的句子。
</div>

<span lang="ja">な</span> "后缀"基本上就是允许 <span lang="ja">な</span>-形容词在句子中起定语作用的东西。你可能已经注意到 \<adj\><span lang="ja">な</span>\<noun\> 结构看起来非常类似于我们在[第5课](./Lesson5.md)中已经见过的 \<noun\><span lang="ja">の</span>\<noun\> 结构。<span lang="ja">な</span> 甚至可以以这种方式用于一些普通名词，不仅仅是 <span lang="ja">な</span>-形容词，但这是一种带有特定细微含义的特殊用法，我们现在不需要担心。

<pre>
<span lang="ja">魔法の世界</span>
魔法的世界

<span lang="ja">魔法な世界</span>
带有魔法感的世界
</pre>

<span lang="ja">魔法</span> 是一个名词（魔法），通常不会这样接 <span lang="ja">な</span>，但一些作者可能会写出这样的东西来赋予它一种特殊的感觉。

回到普通的 <span lang="ja">な</span>-形容词，它们应该被视为与名词不同，词典也确实如此区分。它们明确区分名词（<span lang="ja">名詞</span>）和 <span lang="ja">な</span>-形容词（<span lang="ja">形容動詞</span>）为不同的类别。虽然很多（如果不是大多数）<span lang="ja">な</span>-形容词也是名词，但其中一些不是，不能那样使用。例如，你不能直接将格助词附加到它们上。

<pre>
<span lang="ja">元気</span><b><span lang="ja">な</span></b><span lang="ja">人</span>
精力充沛的人

<span lang="ja">元気</span><b><span lang="ja">が</span></b><span lang="ja">出る</span>
打起精神（直译：精力涌出）

<span lang="ja">静か</span><b><span lang="ja">な</span></b><span lang="ja">人</span>
安静的人

*<span lang="ja">静か</span><b><span lang="ja">が</span></b><span lang="ja">？？？</span>
（不合语法的句子片段）
</pre>

在上面的例子中，<span lang="ja">元気</span> 既可以作为 <span lang="ja">な</span>-形容词，也可以作为名词，但 <span lang="ja">静か</span> 只能作为 <span lang="ja">な</span>-形容词。

一些资源也承认所谓的 <span lang="ja">の</span>-形容词，它们只是一些有点特殊的名词，因为它们可以使用 <span lang="ja">の</span> 而不是 <span lang="ja">な</span> 来转变为类似形容词的含义。

<pre>
<span lang="ja">英語の本</span>
英文书。（不是"英语的书"）

<span lang="ja">普通</span><b><span lang="ja">の</span></b><span lang="ja">人</span>
普通人。
</pre>
