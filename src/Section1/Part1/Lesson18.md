# 使用 <span lang="ja">いる</span> 和 <span lang="ja">ある</span> 表示存在，以及 <span lang="ja">である</span>

动词 <span lang="ja">いる</span> 和 <span lang="ja">ある</span> 表示存在。两者之间的区别在于主语是否是"有生命"的东西。动物和人用 <span lang="ja">いる</span>，但桌子和三明治用 <span lang="ja">ある</span>。像汽车这样会移动但没有生命的东西，有时可以根据上下文和句子的焦点来使用两者之一。不要误以为这关乎"有生命"与否。并不是这样。例如，台风用 <span lang="ja">いる</span>，因为它们自己移动。科幻电影中的机器人也用 <span lang="ja">いる</span>。有时人也可以用 <span lang="ja">ある</span>，特别是在古老的语言和固定短语中。

<pre>
<span lang="ja">家には猫が</span><b><span lang="ja">いる</span></b>
In the house there is a cat.

<span lang="ja">店が</span><b><span lang="ja">ある</span></b>
There is a shop.
</pre>

<div class="warning">
<span lang="ja">ある</span> 和 <span lang="ja">いる</span> 都是存在动词，当要标记存在的位置时，我们通常使用 <span lang="ja">に</span> 而不是 <span lang="ja">で</span>。
</div>

有时，当人们想说明他们拥有某物时，他们使用 <span lang="ja">ある</span> 或 <span lang="ja">いる</span>。它们在技术上不是"拥有"的意思，但这是对我们在英语中使用"have/possess"方式的等价翻译。这只是措辞上的区别。

<pre>
<span lang="ja">デスクが</span><b><span lang="ja">ある</span></b>
There's a desk / I have a desk

<span lang="ja">私の家には猫が</span><b><span lang="ja">いる</span></b>
I have a cat.
</pre>

<span lang="ja">いる</span> 作为一段动词活用。

<div class="warning">
还有另一个 <span lang="ja">いる</span> 动词：<span lang="ja">要る</span>，意思是"需要"。区别在于这个动词是作为五段动词活用的，所以除了基本形之外，你可以很容易地在它们被活用时区分两者。
</div>

<span lang="ja">ある</span> 作为五段动词活用，但其否定活用是不规则的。

|             |
|-------------|
| <span lang="ja">ある</span>        |
| **<span lang="ja">ない</span>**    |
| <span lang="ja">あった</span>      |
| <span lang="ja">あります</span>    |
| <span lang="ja">ありません</span>  |

<pre>
<span lang="ja">店は</span><b><span lang="ja">ない</span></b>
There is no store.
</pre>

现在我们知道 <span lang="ja">ある</span> 了，我们可以提出 <span lang="ja">だ</span> 的起源：<span lang="ja">である</span>。<span lang="ja">である</span> 是 <span lang="ja">だ</span>（是）的正式版本。与 <span lang="ja">だ</span> 和 <span lang="ja">です</span> 不同，<span lang="ja">である</span> 可以用于附着到名词上的关系从句：

<pre>
*<span lang="ja">友達だ人</span>  
(ungrammatical nonsense)

<span lang="ja">友達</span><b><span lang="ja">である</span></b><span lang="ja">人</span>
Person who is a friend (formal)
</pre>

你可能记得我们使用了 <span lang="ja">ではない</span> 作为 <span lang="ja">だ</span> 的否定形式。<span lang="ja">ではない</span> 基本上就是 <span lang="ja">である</span> 的否定形。

<pre>
<span lang="ja">である</span>
<span lang="ja">であった</span>
<span lang="ja">で</span>(<span lang="ja">は</span>)<span lang="ja">ない</span>
</pre>

**注意**：<span lang="ja">でない</span> 也存在，但由于语言的某些特性，添加 <span lang="ja">は</span> 已经成为否定 <span lang="ja">である</span> 时的标准/预期版本。除了"语言就是如此"之外没有真正的原因。你可能仍然会看到 <span lang="ja">でない</span>，尤其是在更正式的书面媒体或某些固定短语中。
