# 使用 <span lang="ja">だ</span> 和 <span lang="ja">です</span> 表示状态

让我们从"A 是 B"形式的句子开始。日语有两个基本的"是"词。语法术语中，这样的词被称为"系词"。系词有专门的术语，因为它们是类似动词的特殊词类。

日语中的两个系词是 <span lang="ja">だ</span> 和 <span lang="ja">です</span>。<span lang="ja">です</span> 比 <span lang="ja">だ</span> 更有礼貌。尽管两者都是系词，但它们的用法可以很灵活，有时也可以省略；你可以在完全省略它们的同时，仍然在句子中表达"A 是 B"类型的关系。尤其是 <span lang="ja">だ</span>，在现代日语中通常被省略在会话句子的末尾。

与英语不同，表示状态的词放在你用来描述某事物的词**之后**，而不是之前。我们会发现，这适用于日语中的大多数修饰词：它们修饰其**左侧**的词。

<pre>
<span lang="ja">ペンだ。</span>
It's a pen.

<span lang="ja">ネコです。</span>
It is a cat.
</pre>

<span lang="ja">だ</span> 和 <span lang="ja">です</span> 之间的区别通常在于礼貌程度和正式程度。需要记住的一个重要点是，虽然 <span lang="ja">です</span> 是 <span lang="ja">だ</span> 的礼貌形式，但两者并不总是可以互换，而且有很多不同的用法。你不能总是随意地将 <span lang="ja">だ</span> 替换为 <span lang="ja">です</span>（反之亦然）。你将在后面的课程中学习其中一些用法。

稍后你还会看到 <span lang="ja">です</span> 用于英语中不允许使用"是"的地方。这是因为它甚至可以充当没有实际含义的填充词。

要表示某事物*不是*，我们将 <span lang="ja">だ</span> 变为 <span lang="ja">ではない</span>，将 <span lang="ja">です</span> 变为 <span lang="ja">ではありません</span>。在口语中，<span lang="ja">では</span> 会被模糊读成 <span lang="ja">じゃ</span>，所以 <span lang="ja">じゃない</span> 和 <span lang="ja">じゃありません</span> 更常见。我们将在后面的课程中了解这个 <span lang="ja">では</span> 的作用。<span lang="ja">ではありません</span> 还有一种替代形式，就是在 <span lang="ja">ではない</span>/<span lang="ja">じゃない</span> 后面加上 <span lang="ja">です</span>，意思是相同的。

<pre>
<span lang="ja">人じゃない</span>
It's not a person

<span lang="ja">本じゃないです</span> or <span lang="ja">本ではありません</span>
It is not a book (polite)
</pre>

日语有两种基本时态：普通形和过去形。由于词典以普通形列出动词，它有时也被称为"词典形"。在此上下文中，"形"和"时态"是可以互换的。

<span lang="ja">だ</span> 的过去形式是 <span lang="ja">だった</span>。<span lang="ja">です</span> 的过去形式是 <span lang="ja">でした</span>。

<pre>
<span lang="ja">オレだった！</span> 
It was me! (casual)

<span lang="ja">そうでした</span>
That was so. (polite)
</pre>

词典形通常被称为现在时或非过去时。重要的是要记住它可以在很多情况下使用。现在时和非过去时是合理的名称，但不要将名称当作规则，有时名称会令人困惑。本指南将交替使用这些名称。

普通形和过去形通常与英语的现在时和过去时对应，但它们并非在所有相同的地方使用。只需记住，英语和日语使用时态的方式常常大不相同。

在日语中，关于未来的普通陈述大多数时候使用普通形。这就是"非过去"这个名称的由来。

有时过去形也可以用于已经完成的事物，如"已经成为"，即使它们是假设性的且尚未发生。你可能也会听到它被称为"完成时"或"完成形"。
