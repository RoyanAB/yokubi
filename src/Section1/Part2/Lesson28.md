# 动词回顾

我们终于学完了所有五段动词的基本词干，所以现在是回顾所学内容的好时机。请不要试图记住这些内容。知道这些概念的存在就足够了。如果你看得眼花缭乱，放轻松。如果做不到，继续往下学就好。

所给出的名称只是对形式的简单描述/提示，并非正式的语法名称。对于想了解的人，我们会同时给出日语名称及其对应的正式术语，但你不需要记住它们。如果在本指南的后续内容中提到某个词干或形式，会用简洁的英文描述，而不是日语名称。

| 形式             | 名称                | 日语术语                                          |
|-----------------|---------------------|--------------------------------------------------|
| <span lang="ja">書</span>**<span lang="ja">か</span>**(<span lang="ja">ない</span>)  | 否定词干              | <span lang="ja">未然形</span> - 未然形（irrealis form）                  |
| <span lang="ja">書</span>**<span lang="ja">き</span>**(<span lang="ja">ます</span>)  | ます词干              | <span lang="ja">連用形</span> - 连用形（continuative form）              |
| <span lang="ja">書</span>**<span lang="ja">く</span>**        | 普通形/词典形          | <span lang="ja">終止形</span> 终止形（predicative）或<span lang="ja">連体形</span> 连体形（attributive form）  |
| <span lang="ja">書</span>**<span lang="ja">け</span>**(<span lang="ja">る</span>/<span lang="ja">ば</span>) | 可能词干              | <span lang="ja">已然形</span> 已然形（perfective）或<span lang="ja">仮定形</span> 假定形（hypothetical form）  |
| <span lang="ja">書</span>**<span lang="ja">け</span>**        | 命令形                | <span lang="ja">命令形</span> - 命令形（imperative form）                |
| <span lang="ja">書</span>**<span lang="ja">こう</span>**      | 意志形                | <span lang="ja">意向形</span> - 意向形（volitional form）                |

请记住，对于五段动词，过去形和<span lang="ja">て</span>形有特殊的变化规则。例如，k行中<span lang="ja">き</span>变成<span lang="ja">い</span>：

<pre>
<span lang="ja">書</span><b><span lang="ja">い</span></b><span lang="ja">て</span>
<span lang="ja">書</span><b><span lang="ja">い</span></b><span lang="ja">た</span>
</pre>

对于一段动词，词干保持不变，因此简单得多，只需替换<span lang="ja">る</span>即可：

<pre>
<span lang="ja">食べ</span>(<span lang="ja">ない</span>) 否定形
<span lang="ja">食べ</span>(<span lang="ja">ます</span>) ます形
<span lang="ja">食べる</span> 普通形/词典形
<span lang="ja">食べ</span>(<span lang="ja">られる</span>/<span lang="ja">れる</span>) 可能形
<span lang="ja">食べ</span>(<span lang="ja">れば</span>) 条件形/<span lang="ja">ば</span>形
<span lang="ja">食べ</span>(<span lang="ja">ろ</span>) 命令形
<span lang="ja">食べ</span>(<span lang="ja">よう</span>) 意志形
<span lang="ja">食べ</span>(<span lang="ja">て</span>) <span lang="ja">て</span>形
<span lang="ja">食べ</span>(<span lang="ja">た</span>) 过去形
</pre>

以及两个不规则动词<span lang="ja">する</span>和<span lang="ja">くる</span>：

| 形式        | <span lang="ja">する</span>      | <span lang="ja">くる</span>     |
|------------|-----------|----------|
| 否定形       | <span lang="ja">しない</span>    | <span lang="ja">こない</span>   |
| ます词干     | <span lang="ja">し</span>(<span lang="ja">ます</span>)  | <span lang="ja">き</span>(<span lang="ja">ます</span>) |
| 普通形       | <span lang="ja">する</span>      | <span lang="ja">くる</span>     |
| 可能形       | <span lang="ja">できる</span>    | <span lang="ja">これる</span>   |
| <span lang="ja">ば</span>词干   | <span lang="ja">すれば</span>    | <span lang="ja">くれば</span>   |
| 命令形       | <span lang="ja">しろ</span>/<span lang="ja">せよ</span> | <span lang="ja">こい</span>     |
| 意志形       | <span lang="ja">しよう</span>    | <span lang="ja">こよう</span>   |
| <span lang="ja">て</span>形    | <span lang="ja">して</span>      | <span lang="ja">きて</span>     |
| 过去形       | <span lang="ja">した</span>      | <span lang="ja">きた</span>     |

另外，提醒一下，五段动词<span lang="ja">行く</span>的<span lang="ja">て</span>形和过去形的活用是不规则的，分别为<span lang="ja">行って</span>/<span lang="ja">行った</span>。
