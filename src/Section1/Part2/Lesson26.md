# 用<span lang="ja">たい</span>和<span lang="ja">ほしい</span>表达意愿和愿望

在本课中，我们将学习三种用日语表达愿望或意图的方式。

"意志形"，即<span lang="ja">おう</span>/<span lang="ja">よう</span>，表示你想要或将会做某事，或者邀请他人表达意愿，如"让我们做X"。它也用于关于意志动作的复合结构中。五段动词的意志形是 -ou，一段动词的意志形是<span lang="ja">よう</span>。它读作"长音o"，而不是<span lang="ja">お</span>加<span lang="ja">う</span>。

<pre>
<span lang="ja">行こう</span>
走吧。

<span lang="ja">行こうか</span>
我们去怎么样？

<span lang="ja">食べようと思う</span>
我想我要吃了。
</pre>

注意：我们将在后面的章节中学习<span lang="ja">と</span>与<span lang="ja">思う</span>的这种用法，现在不用太担心。

<span lang="ja">です</span>的意志形是<span lang="ja">でしょう</span>，<span lang="ja">だ</span>的意志形是<span lang="ja">だろう</span>。这些通常用来"邀请一个想法"，比如推测或可能性。

<pre>
<span lang="ja">いいでしょう</span>
挺好的，你不觉得吗？

<span lang="ja">犬だろうと、猫だろうと、なんでも問題ありません</span>   
也许是狗，也许是猫；完全没问题。
</pre>

第二种表达愿望的方式是<span lang="ja">たい</span>。<span lang="ja">たい</span>是一个类似<span lang="ja">い</span>形容词的助动词，它的接续方式与<span lang="ja">ます</span>相同。

<pre>
<span lang="ja">死にたい</span>
我想死。

<span lang="ja">アイスが食べたい</span>
我想吃冰淇淋。
</pre>

在最后一个例句中，用<span lang="ja">を</span>也是可以的。关于<span lang="ja">が</span>和<span lang="ja">を</span>，<span lang="ja">たい</span>的工作方式与可能动词类似。<span lang="ja">が</span>或<span lang="ja">を</span>都可以用来标记愿望的对象/目标，尽管不同的动词可能对其中一个有特定的偏好。不要试图去记住这个，只需知道它的存在且是正确的日语即可。

最后一种表达愿望的方式是使用<span lang="ja">欲しい</span>，一个<span lang="ja">い</span>形容词。

<pre>
<span lang="ja">アイスがほしい</span>
我想要冰淇淋
</pre>

注意：与<span lang="ja">たい</span>类似，<span lang="ja">ほしい</span>也属于那一类特殊的形容词，其中<span lang="ja">が</span>标记愿望的对象，有时可以被<span lang="ja">を</span>替代。然而，与<span lang="ja">たい</span>不同的是，<span lang="ja">をほしい</span>远不如前者规范，也更不常见。你可能仍然会看到它，但很多人会觉得这是别扭的日语。最好避免使用它。

<span lang="ja">ほしい</span>的另一个特殊之处在于，它可以在动词<span lang="ja">て</span>形之后用作辅助动词，以发出柔和的命令或请求。

<pre>
<span lang="ja">その話はやめてほしい</span>
我希望你不要再说那个了
</pre>

这可能听起来比<span lang="ja">てください</span>更客观和间接，也可以用来从你的角度为某人提供一般性的建议。需要记住的重要一点是，你使用<span lang="ja">がほしい</span>来谈论自己，使用<span lang="ja">てほしい</span>来谈论他人。
