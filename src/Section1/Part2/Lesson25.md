# 可能形与<span lang="ja">できる</span>

可能形的存在是为了让人们能够表达某事"可以做到"。对于五段动词，将u音变为e音，然后加上<span lang="ja">る</span>。所以"u" -> "eru"。对于一段动词，可能形的变化与被动形相同。是的，这很令人困惑。正因如此，有时你可能会遇到一些句子，其中一段动词到底是被动形还是可能形令人费解/故意含糊不清。你只能凭感觉来理解。

<pre>
<span lang="ja">食べ</span><b><span lang="ja">られる</span></b>（或<span lang="ja">食べ</span><b><span lang="ja">れる</span></b>（！））
能吃

<span lang="ja">殺</span><b><span lang="ja">せる</span></b>
能杀

<span lang="ja">死</span><b><span lang="ja">ねる</span></b>
能死

<span lang="ja">飲</span><b><span lang="ja">める</span></b>
能喝
</pre>

你可能已经注意到，<span lang="ja">食べれる</span>被列为<span lang="ja">食べられる</span>的可接受变体，旁边还标注了一个大大的**！**。在日常口语/俗语日语中，通过去掉<span lang="ja">ら</span>音来将一段动词的可能形缩短是很常见的（这被称为<span lang="ja">ら抜き</span>）。这不是"规范"的用法，不应在正式场合或书面语中使用，但在日常口语中相当常见。并非所有一段动词都可以这样用，但没有明确的规则规定哪些可以哪些不可以。这取决于个人偏好，所以不要试图去学这个规则。只需知道它的存在，并学会识别它即可。

对于不规则动词<span lang="ja">する</span>，我们使用替代动词<span lang="ja">できる</span>作为其可能形的变体。这是一个例外，需要单独记住。

<pre>
<span lang="ja">昨日は勉強</span><b><span lang="ja">できなかった</span></b>
我昨天<b>没能</b>学习
</pre>

为了让可能形/被动形的二重性更加令人困惑，我们需要谈谈可能动词在日语中带有的一个特殊现象，这是英语中所没有的。

在可能形中，他动词的"主语"和"宾语"概念在语法上可能变得模糊不清。让我们来看一个例子：

<pre>
<span lang="ja">私</span><b><span lang="ja">は</span></b><span lang="ja">ピザ</span><b><span lang="ja">を</span></b><span lang="ja">食べる</span>
我吃披萨（非可能形，标准形式）

<span lang="ja">私</span><b><span lang="ja">は</span></b><span lang="ja">ピザ</span><b><span lang="ja">が</span></b><span lang="ja">食べられる</span>
<span lang="ja">私</span><b><span lang="ja">に</span></b><span lang="ja">ピザ</span><b><span lang="ja">が</span></b><span lang="ja">食べられる</span>
<span lang="ja">私</span><b><span lang="ja">は</span></b><span lang="ja">ピザ</span><b><span lang="ja">を</span></b><span lang="ja">食べられる</span>
我能吃披萨（可能形，标准形式）
</pre>

这里到底是怎么回事？这三种选项都是有效的，而且意思基本相同。此外，根据上下文和周围短语的不同，这些例子中的<span lang="ja">は</span>也可以变成<span lang="ja">が</span>（是的，甚至可以有<span lang="ja">私がピザが食べられる</span>，一个双<span lang="ja">が</span>的句子）。

你所需要记住的是，在可能动词中，<span lang="ja">が</span>可以用来**也**标记动作的宾语，而不仅仅是主语。而且在这些句子中，主语有时可以由<span lang="ja">に</span>来标记。有些人认为像上面例子中用<span lang="ja">を</span>是不规范的，然而这极其主观，而且你常常会看到它，以至于认为它不规范只是老头子们对云朵大喊大叫罢了。<span lang="ja">を</span>的版本在从句或非常长且复杂的句子中往往更常见。

还有另一种构成具有可能含义的短语的方法，那就是简单地在动词普通形后面加上<span lang="ja">ことができる</span>。我们将在后面的课程中学习<span lang="ja">こと</span>的含义，但现在你应该将<span lang="ja">ことができる</span>视为一个整体。

<pre>
<span lang="ja">俺は答える</span><b><span lang="ja">ことができなかった</span></b><span lang="ja">。</span>
我<b>没能</b>回答。
</pre>

<span lang="ja">ことができる</span>与普通可能形变化之间的区别非常小，通常难以察觉。<span lang="ja">ことができる</span>可能因为更冗长而显得稍微正式一些，但这并不值得太过担心。你可以认为它们基本上是可以互换的。
