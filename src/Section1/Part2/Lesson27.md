# 用四种条件形表达"如果"和"当……时"

日语条件句中先陈述条件，然后陈述结果动作。条件句是我们可以将两个"A做B"类型的句子连接起来的方式之一，结构如"如果A做B，那么C做D"。当然，"如果A是B，那么C做D"或任何其他组合结构也都是可以的。

要是真那么简单，只用一种条件形就能应付，那该多好啊，但日语的有趣之处在于有很多不同的方式来表达相似的事情，而条件形正是其中之一。不幸的是，它们有不同的用法和语气，非常难以用英语语法来解释，所以你必须依靠直觉和接触来掌握。

主要有四种条件形：<span lang="ja">たら</span>、<span lang="ja">ば</span>、<span lang="ja">と</span>和<span lang="ja">なら</span>。它们都分布在"如果"和"当……时"甚至"随着"（如在"As I woke up, I heard a weird sound outside"中）的频谱上。实际上还有更多构成条件句的方式，但这些是本课的主要内容。

一如既往，不要试图完美记住本课内容，只需学会在你遇到每种例子时识别它们，并大体上知道它们表示"如果"或"当……时"即可。

我们将看到的最简单的条件形是<span lang="ja">ば</span>形，它将动词变为一个条件。其变化形式是将动词末尾的u变为e，然后加上<span lang="ja">ば</span>。基本上，对于五段动词，你将u变为e，对于一段动词，你在词干后加上<span lang="ja">れ</span>。

<pre>
<span lang="ja">見</span><b><span lang="ja">れば</span></b><span lang="ja">死ぬ</span>
<b>如果</b>看的话你会死的。

<span lang="ja">お前を殺せ</span><b><span lang="ja">ば</span></b><span lang="ja">、すべてが終わる。</span>
<b>如果</b>杀了你，一切就会结束。
</pre>

<span lang="ja">ば</span>暗示条件是结果发生的充分条件。它是一个真正的逻辑"if"。这甚至可以用来表示某些请求，例如"如果你迟到了，就打电话，我会处理好一切的"。然而，<span lang="ja">ば</span>并不暗示条件最终会发生，因此不能用于像"When you come around, come see me"这样的事情中。在叙述中，它有时可以表示"X发生了，然后Y"，但这种非条件用法相当有限，在文学作品之外并不常见。

对于系词<span lang="ja">だ</span>的<span lang="ja">ば</span>条件形，我们需要将其"展开"为<span lang="ja">である</span>，然后对此进行变化。

<pre>
<span lang="ja">普通の人</span><b><span lang="ja">であれば</span></b><span lang="ja">、その命令に逆らうことはできないだろう</span>
如果他是一个普通人，想必无法违抗那道命令吧。
</pre>

对于<span lang="ja">い</span>形容词，我们将<span lang="ja">い</span>变成<span lang="ja">ければ</span>：

<pre>
<span lang="ja">良</span><b><span lang="ja">ければ</span></b><span lang="ja">教えてください。</span>
如果你觉得可以的话，请告诉我。
</pre>

<span lang="ja">たら</span>是另一种在某些用法上与<span lang="ja">ば</span>相似的条件形。然而，与<span lang="ja">ば</span>不同，它有更广泛的范围和顺序性动作序列的含义。它可以在纯粹假设的序列或严格时间顺序的序列中连接两个事件。它可以是"如果A发生，那么B"，也可以是"当A发生时，B紧接着发生"。为了更容易区分这种区别，假设性用法中通常会用<span lang="ja">もし</span>作为开头词来提示听者。

<pre>
<b><span lang="ja">もし</span></b><span lang="ja">敵がい</span><b><span lang="ja">たら</span></b><span lang="ja">私が斬り捨てます</span>
<b>如果</b>有敌人，我会将他们斩杀（假设）

<span lang="ja">朝起き</span><b><span lang="ja">たら、</span></b><span lang="ja">食堂には誰もいなかった。</span>
<b>早上醒来时</b>，食堂里一个人都没有。
</pre>

<span lang="ja">たら</span>的"当……时"用法在叙述过去发生的"一次性<span lang="ja">”</span>事件时很常见，通常是意外发生的事件。

在假设性用法中，<span lang="ja">たら</span>和<span lang="ja">ば</span>通常可以互换。甚至存在地区差异，某些日本地区的人可能更偏好其中一个，反之亦然。这就是为什么你不应该问"为什么这个人用了<span lang="ja">たら</span>而不是<span lang="ja">ば</span>？"这类问题。接受它就对了。

<span lang="ja">たら</span>还有一个特殊用法，通常是直接跟在名字后面，以<span lang="ja">ったら</span>的形式出现。这个<span lang="ja">ったら</span>有着完全不同的含义，只是作为表示恼怒感叹的助词。它不是条件形。

<pre>
<span lang="ja">もう、お父さん</span><b><span lang="ja">ったら</span></b><span lang="ja">、しっかりしなさい！</span>
真是的，老爸，你给我振作一点！
</pre>

<span lang="ja">と</span>用于连接两个事件，其中一个事件是另一个事件的自然结果，具有强烈的顺序性。这类似于<span lang="ja">たら</span>的时间顺序性，但<span lang="ja">と</span>不是标记个别发生的事件（"我做了X然后……"），而是用来描述每次都会发生的普遍事件（"一旦你做了X，那么……"）。

<pre>
<span lang="ja">あんまり食べる</span><b><span lang="ja">と</span></b><span lang="ja">太りますよ</span>
<b>如果</b>吃太多，会变胖的哦
</pre>

这个<span lang="ja">と</span>也常用于叙述中，描述一个接一个发生的一系列事件。在这种用法中，它使人感觉读者无法控制这些事件，他们只是一个旁观者，看着这些事件在眼前"自然地"发生。

<pre>
<span lang="ja">少し歩く</span><b><span lang="ja">と</span></b><span lang="ja">、開けた場所に出た。</span>
<b>走了一会儿后</b>，他们来到了一片开阔地带。
</pre>

本课的最后一种条件形是<span lang="ja">なら</span>。<span lang="ja">なら</span>是一种严格的假设条件形。它暗示在该陈述成立的假设世界中，条件被假定为真。
这意味着当你说 X <span lang="ja">なら</span> Y 时，意思接近于"如果我们考虑X为真这种情况，那么Y"，但它在日语中听起来并不像英语这么冗长。这是一种非常常见的表达方式。

<pre>
<span lang="ja">行きたい</span><b><span lang="ja">なら</span></b><span lang="ja">一緒に行こう</span>
<b>如果你想的话</b>，我们一起走吧
</pre>

由于<span lang="ja">なら</span>的这一特殊语气，它是唯一一种第二个分句可以在第一个分句之前发生的条件形。<span lang="ja">と</span>、<span lang="ja">たら</span>和<span lang="ja">ば</span>都不能做到这一点。

<pre>
<span lang="ja">運転する</span><b><span lang="ja">なら</span></b><span lang="ja">飲まないで</span>
<b>如果你要</b>开车的话，就别喝酒。
</pre>

<span lang="ja">飲む</span>的动作必须在<span lang="ja">運転する</span>这个条件之前发生，因此说话者**假设**<span lang="ja">運転する</span>会在该建议（<span lang="ja">飲まないで</span>）生效的假设世界中发生。

有时<span lang="ja">なら</span>会以<span lang="ja">ならば</span>的形式出现。意思相同，但感觉更书面化。

对于动词和<span lang="ja">い</span>形容词，有时可以在动词和<span lang="ja">なら</span>之间加上<span lang="ja">の</span>。这可以带来额外的语气（类似于<span lang="ja">のだ</span>中的说明性<span lang="ja">の</span>），但你不需要记住这一点。只需记住带<span lang="ja">の</span>的版本可以听起来更书面和正式即可。

<pre>
<span lang="ja">そんなに知りたい</span><b><span lang="ja">のなら</span></b><span lang="ja">、教えてあげましょう。</span>
<b>既然</b>你这么想知道，那我就告诉你吧。

<span lang="ja">そんなに知りたい</span><b><span lang="ja">なら</span></b><span lang="ja">教えてやる。</span>
<b>既然</b>你这么想知道，我告诉你
</pre>
