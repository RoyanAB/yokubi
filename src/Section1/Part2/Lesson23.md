# 用<span lang="ja">こそあど</span>词指示

指示词是指表示"这个"、"那个"等含义的词。

日语在这方面的不同之处有两点。首先，它的指示词更加规则化，其次，它的指示词对应的是三个"位置"而不是两个。这用例子来说明比用文字解释更容易，让我们来看例子：

<pre>
<b><span lang="ja">こ</span></b><span lang="ja">れ</span>
这个（靠近说话者的东西）

<b><span lang="ja">そ</span></b><span lang="ja">れ</span>
那个（靠近听者的东西）

<b><span lang="ja">あ</span></b><span lang="ja">れ</span>
那个（离说话者和听者都远的东西）

<b><span lang="ja">ど</span></b><span lang="ja">れ</span>
哪个

<b><span lang="ja">これ</span></b><span lang="ja">はペンです</span>
<b>这</b>是一支笔

<b><span lang="ja">それ</span></b><span lang="ja">は猫ですか？</span>
<b>那</b>是一只猫吗？（猫靠近听者）

<b><span lang="ja">あれ</span></b><span lang="ja">は何？</span>
<b>那</b>是什么？（指着离你和听者都远的东西）

<b><span lang="ja">どれ</span></b><span lang="ja">がいい？</span>  
<b>哪个</b>好？
</pre>

以上四个词是"指示代词"。其中有两个在英语里都翻译成"that"，但在日语中它们有不同含义。<span lang="ja">それ</span>用于靠近听者的东西。<span lang="ja">あれ</span>用于离说话者和听者都远的东西。

如果你还没注意到其中的规律，正如本课标题所指出的，我们有时将它们合称为<span lang="ja">こそあど</span>，因为它们之间只差第一个假名音节。

尽管两种"that"之间存在区别，但这些词的使用方式与英语中的使用方式非常相似。

还有另一组非常相似的<span lang="ja">こそあど</span>指示词，它们在语法上作用不同，但翻译成英语时很难解释，所以我们先来看例子：

<pre>
<span lang="ja">この</span>&lt;名词&gt;
这个&lt;名词&gt;

<span lang="ja">その</span>&lt;名词&gt;
那个&lt;名词&gt;

<span lang="ja">あの</span>&lt;名词&gt;
那个&lt;名词&gt;（远处的）

<span lang="ja">どの</span>&lt;名词&gt;
哪个&lt;名词&gt;
</pre>

这些词不能单独存在，它们**必须**附着在后面的名词上。意思与另一组相同，但用法略有不同。

想象一下你手里拿着一本书，你想说"这是一本书"。"书"是句子的焦点，所以你会说<span lang="ja">これは本です</span>。你只是简单地使用系词在陈述"this = book"。然而，如果你想表达关于**这本书**的某些内容，你会改为说<span lang="ja">この本は面白いです</span>。**这本书**很有趣。"这个"和"书"变成了一个整体。在英语中，"This is a book"和"This book is interesting"使用相同的"this"，但在日语中，一个用<span lang="ja">これ</span>，另一个用<span lang="ja">この</span>。

同样的情况也适用于所有带<span lang="ja">の</span>的其他<span lang="ja">こそあど</span>词。

还有许多其他属于<span lang="ja">こそあど</span>结构的词。我们不会在这里全部讲解，但它们通常相当规律，因为它们都遵循以下规则：

<pre>
<span lang="ja">こ</span> = 靠近说话者  
<span lang="ja">そ</span> = 靠近听者  
<span lang="ja">あ</span> = 距离两者都远  
<span lang="ja">ど</span> = 疑问词
</pre>

<span lang="ja">あ</span>是唯一发音稍有不规律的行，但当你遇到它们时就会知道。以下是一些例子：

|                   |                    |            |         |
|-------------------|--------------------|------------|---------|
| <span lang="ja">ここ</span>              | <span lang="ja">そこ</span>               | <span lang="ja">あそこ</span>     | <span lang="ja">どこ</span>    |
| 这里（这个地方）    | 那里（那个地方）     | 那边        | 哪里？   |

<br />

|                 |                 |                |                  |
|-----------------|-----------------|----------------|------------------|
| <span lang="ja">こちら</span>          | <span lang="ja">そちら</span>          | <span lang="ja">あちら</span>         | <span lang="ja">どちら</span>           |
| 这个方向          | 那个方向          | 那个方向         | 哪个方向？         |

<br />

|                 |                |                |                  |
|-----------------|----------------|----------------|------------------|
| <span lang="ja">こう</span>            | <span lang="ja">そう</span>           | <span lang="ja">ああ</span>           | <span lang="ja">どう</span>             |
| 以这种方式        | 以那种方式       | 以那种方式       | 以哪种方式？       |

<br />

|           |           |            |            |
|-----------|-----------|------------|------------|
| <span lang="ja">こいつ</span>    | <span lang="ja">そいつ</span>    | <span lang="ja">あいつ</span>     | <span lang="ja">どいつ</span>     |
| 这家伙      | 那家伙      | 那家伙       | 哪个家伙     |

注意：<span lang="ja">こちら</span>组也可以用作在正式场合指代人的礼貌方式，因为它增加了间接性。另一方面，<span lang="ja">こいつ</span>组通常听起来很粗鲁，所以不要用<span lang="ja">こいつ</span>来称呼人。

就像在英语中一样，物理距离并不是这些词的唯一决定因素。它们也可以用来指代时间上较远的事物。它们甚至可能受到说话者和听者对话题熟悉程度的影响。对话中较早提到的东西可能被称为<span lang="ja">あの</span>\<事物\>，但刚刚提到的东西更可能被称为<span lang="ja">その</span>\<事物\>。就像在英语中我们有时不确定应该说"this"还是"that"一样，即使在日语中，人们也不总是意见一致，所以在这些情况下，区分并不总是严格的。
