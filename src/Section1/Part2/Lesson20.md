# 用<span lang="ja">の</span>提问与说明性的<span lang="ja">のだ</span>

另一个与<span lang="ja">か</span>类似的疑问助词是<span lang="ja">の</span>。用<span lang="ja">の</span>提问带有一种与用<span lang="ja">か</span>提问相当不同的含义。我们稍后会解释其语气的含义，先来关注结构。

要用<span lang="ja">の</span>提问，在普通形中，你只需在动词和<span lang="ja">い</span>形容词后面加<span lang="ja">の</span>，在名词和<span lang="ja">な</span>形容词后面加<span lang="ja">なの</span>（不加<span lang="ja">だ</span>）。问号是可选的。

|                 |                  |
|-----------------|------------------|
| <span lang="ja">学生</span>(<span lang="ja">だ</span>)        | <span lang="ja">学生なの？</span>       |
| 他是学生         | 他是学生吗？      |

<br />

|              |               |
|--------------|---------------|
| <span lang="ja">普通</span>(<span lang="ja">だ</span>)     | <span lang="ja">普通なの？</span>    |
| 这是正常的     | 这正常吗？      |

<br />

|            |             |
|------------|-------------|
| <span lang="ja">寒い</span>       | <span lang="ja">寒いの？</span>    |
| 很冷         | 冷吗？         |

<br />

|            |                |
|------------|----------------|
| <span lang="ja">魚を食べる</span> | <span lang="ja">魚を食べるの？</span> |
| 我吃鱼       | 你吃鱼吗？       |

在礼貌形中，我们只需在<span lang="ja">の</span>后面加上<span lang="ja">ですか</span>，即使是动词句也是如此。在日常口语中，这通常会被连读为<span lang="ja">んですか</span>而不是<span lang="ja">のですか</span>。

| 普通形           | 礼貌形                 |
|----------------|----------------------|
| <span lang="ja">学生なの？</span>     | <span lang="ja">学生なんですか？</span>     |
| <span lang="ja">普通なの？</span>     | <span lang="ja">普通なんですか？</span>     |
| <span lang="ja">寒いの？</span>       | <span lang="ja">寒いんですか？</span>       |
| <span lang="ja">魚を食べるの？</span> | <span lang="ja">魚を食べるんですか？</span> |

<span lang="ja">の</span>也可以用于陈述句结尾，而非疑问句。在这种用法中，语调应该能清楚表明这不是一个疑问句，同时结构也稍有不同。

要构成<span lang="ja">の</span>陈述句，在普通形日语中，你遵循与<span lang="ja">の</span>疑问句相同的规则，但以<span lang="ja">のだ</span>（或<span lang="ja">んだ</span>）结尾：

<pre>
<span lang="ja">学生なんだ</span>
是学生

<span lang="ja">普通なんだ</span>
是正常的

<span lang="ja">寒いんだ</span>
很冷

<span lang="ja">魚を食べるんだ</span>
我吃鱼
</pre>

在礼貌形中，与<span lang="ja">のですか</span>疑问句一样，只需去掉<span lang="ja">です</span>后面的<span lang="ja">か</span>即可。

<pre>
<span lang="ja">学生なんです</span>

<span lang="ja">普通なんです</span>

<span lang="ja">寒いんです</span>

<span lang="ja">魚を食べるんです</span>
</pre>

以<span lang="ja">だ</span>结尾的陈述句可能听起来很生硬，人们常常倾向于在普通形陈述句的<span lang="ja">んだ</span>后面加上<span lang="ja">よ</span>或<span lang="ja">ね</span>来缓和语气。我们将在后面的章节中学习<span lang="ja">よ</span>和<span lang="ja">ね</span>这两个句末助词。

你也可以直接用<span lang="ja">の</span>结束<span lang="ja">の</span>陈述句，而不加<span lang="ja">だ</span>，但这种说法听起来带有女性气质，要小心使用。

现在我们来谈谈这些<span lang="ja">の</span>疑问句和陈述句所传达的语气。

从表面上看，<span lang="ja">の</span>疑问句和<span lang="ja">のだ</span>陈述句与不带<span lang="ja">の</span>(<span lang="ja">だ</span>)的普通疑问句和陈述句看起来很相似，但<span lang="ja">の</span>(<span lang="ja">だ</span>)的存在给它们增添了某种"解释性"或"询问性"的语气。

当你用<span lang="ja">の？</span>或<span lang="ja">んですか？</span>提问时，你是在暗示你期望对方对你所问的内容提供解释、理由或说明。你通常在对某事寻求澄清时会使用这种疑问句。

<pre>
<span lang="ja">メアリーさんは学生なんですか？</span>
莫非玛丽是学生？
</pre>

当你在对话中通过一些间接的证据得出玛丽可能是学生的结论，但又不确定时，你可能会这样问。你是在寻求确认。

<pre>
<span lang="ja">猫が好きなんですか？</span>
莫非你喜欢猫？
</pre>

当你看别人开心地和猫玩耍，想确认一下时，你可能会这样问。

为了让区别更加清楚，我们来比较这两个句子：

<pre>
<span lang="ja">何をしてる？</span>
你在做什么？

<span lang="ja">何をしてるの？</span>
你到底在做什么？
</pre>

注意：第二个句子中用斜体强调表示英语中的语气。

你可以想象第一个句子是你在电话里问朋友在做什么，只是因为好奇、想找个话题聊天。然而，第二个听起来更有追问的意味，甚至可能带有攻击性，你可以想象你当场抓到有人正在撬你的车时用的就是这种语气。就像"喂，你到底在干什么？"你是在期待，甚至可以说是要求对方给出解释。

类似地，当我们用解释性的语气做陈述时，会使用<span lang="ja">んだ</span>。就像用<span lang="ja">の？</span>提问是在寻求澄清一样，用<span lang="ja">んだ</span>做陈述则是在提供说话者希望让听者知道的解释或补充信息。

<span lang="ja">んだ</span>有一些常见的用法，但我们不会在这里全部讲解，因为这是一个非常微妙的主题，很难解释，也超出了初学者指南的范围。通过大量的沉浸式接触，你会看到它们被大量使用。只需了解这个基本区别：

<pre>
<span lang="ja">私は日本人です</span>
我是日本人

<span lang="ja">私は日本人なんです</span>
其实，我是日本人（你可能不知道，但我是日本人）
</pre>

前者听起来像是一个简单的陈述，平铺直叙。你是在陈述关于你自己的一个事实。而后者听起来像是在解释什么，或者试图澄清某种情况。它的具体含义取决于上下文，但传达出来的语气是，你觉得有必要通过陈述这个事实来提供解释或澄清。

<pre>
<span lang="ja">世の中にはいろんなことがある</span><b><span lang="ja">んだ</span></b>
这个世界上有各种各样的事情（我告诉你）

<span lang="ja">それからのことはよく覚えていない</span><b><span lang="ja">んだ</span></b>
那之后的事我不太记得了（我是在为自己解释）

<span lang="ja">そんなことはどうでもいい</span><b><span lang="ja">んだ</span></b><span lang="ja">よ</span>
我都说了我不在乎那种事情了（用解释性语气加强强调）
</pre>

<span lang="ja">んだ</span>有一个值得指出的特定用法是，它经常用于说话者意识到某些新信息的情况。这通常使用过去时态，以突出从"不知道"到"知道"的状态转变。

<pre>
<span lang="ja">へー、学生</span><b><span lang="ja">だったんだ</span></b>
哦，原来是学生啊。

<span lang="ja">ああ、そういうこと</span><b><span lang="ja">だったんだ</span></b>
啊，原来是这么回事啊。（直译："啊，是那种事情啊"）
</pre>

它也可以作为一种固定短语，用来告诉对方你正在跟上对话。

<pre>
<span lang="ja">ふーん、そう</span><b><span lang="ja">なんだ</span></b>
哦，原来如此。
</pre>

通常，用<span lang="ja">のだ</span>结尾的陈述句来回答用<span lang="ja">の</span>提出的疑问句是很常见的。

注意：有时你可能会看到<span lang="ja">の</span>疑问句以<span lang="ja">のだ</span>结尾，句末通常带有<span lang="ja">？</span>，语调会清楚表明这是一个疑问句。这与普通的<span lang="ja">の</span>疑问句没有太大区别，但听起来更强硬，可能更有攻击性：<span lang="ja">何してるんだ？！</span>
