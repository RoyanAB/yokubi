# 用<span lang="ja">ている</span>和<span lang="ja">てある</span>表达持续的动作和状态

现在让我们来看看<span lang="ja">ている</span>和<span lang="ja">てある</span>。它们使用<span lang="ja">いる</span>/<span lang="ja">ある</span>作为辅助动词。

<span lang="ja">ている</span>表示持续的动作或状态。对于很多动词来说，这意味着进行或持续（持续的动作），但对于另一些动词来说，它表示一个动作的结果保持在某种状况中（持续的状态）。

<pre>
<span lang="ja">食べ</span><b><span lang="ja">ている</span></b> 
我在吃。（持续的动作）

<span lang="ja">死ん</span><b><span lang="ja">でいる</span></b>
他死了。（持续的状态）
</pre>

<span lang="ja">ている</span>中的<span lang="ja">い</span>在日常对话中通常会被省略，无论是在书写还是口语中都是如此。这包括<span lang="ja">ます</span>形也一样（变成<span lang="ja">てます</span>）。

<pre>
<span lang="ja">食べ</span><b><span lang="ja">てる</span></b>
我在吃。
</pre>

<span lang="ja">ておる</span>是<span lang="ja">ている</span>的正式变体。<span lang="ja">おる</span>和<span lang="ja">いる</span>的意思大致相同。<span lang="ja">ておる</span>在口语中可以被说成<span lang="ja">とる</span>。

<pre>
<span lang="ja">何を言っ</span><b><span lang="ja">ておる</span></b><span lang="ja">のだ。</span>
你在说什么
</pre>

<span lang="ja">てある</span>是关于某事物处于由某人的动作所导致的持续状态。<span lang="ja">てある</span>总是表达一个结果状态，**从不**表示一个动作。

<pre>
<span lang="ja">晩ご飯はもう作っ</span><b><span lang="ja">てある</span></b>
晚饭已经做好了
</pre>

<span lang="ja">てある</span>可以把直接宾语变成主语。这意味着动词原本的"逻辑宾语"由<span lang="ja">が</span>或<span lang="ja">は</span>（或不标记）来标记，就像上面例句中的晚饭一样。不过，由于日语在语言学上有些奇特，有时也可以使用<span lang="ja">を</span>。

最后，它表达的是一个状态而非动作，所以<span lang="ja">てある</span>不应被视为动词的"被动"版本。日语真正的被动形我们很快就会学到。
