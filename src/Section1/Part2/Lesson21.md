# 用<span lang="ja">も</span>和<span lang="ja">と</span>表达更多含义

<span lang="ja">も</span>是一个包含标记。你可以把它理解为"也"或"还"。有时也可以理解为"甚至"。

<span lang="ja">も</span>是一个助词，类似于<span lang="ja">は</span>，会与其他已有的助词一起使用。当<span lang="ja">も</span>用于标记通常由<span lang="ja">が</span>或<span lang="ja">を</span>标记的内容时，原助词会被省略，只保留<span lang="ja">も</span>。与其他助词一起使用时，你只需在原助词后面加上<span lang="ja">も</span>即可。

<pre>
<span lang="ja">私</span><b><span lang="ja">も</span></b><span lang="ja">行く</span>
我<b>也</b>去（除了别人以外）

<span lang="ja">魚</span><b><span lang="ja">も</span></b><span lang="ja">食べたい</span>
我<b>也</b>想吃鱼（除了别的东西以外）

<span lang="ja">学校</span><b><span lang="ja">にも</span></b><span lang="ja">行く</span>
我<b>也</b>去学校（我也去别的地方，但也去学校）
</pre>

<div class="warning">
虽然<span lang="ja">も</span>通常会"覆盖"<span lang="ja">を</span>，但你也可能会遇到<span lang="ja">をも</span>。这比较少见，不值得过多关注，只是见到时不要惊讶。你可以把它理解为一种更强调的版本。
</div>

<span lang="ja">も</span>可以与否定句一起使用。通常，它把给定的名词包含在否定句成立的事物集合中，但有时会因为否定改变了逻辑包含/排除的方式而产生一些奇怪的用法。

<pre>
<span lang="ja">僕</span><b><span lang="ja">も</span></b><span lang="ja">食べない</span>
我<b>也</b>不吃（除了别人以外）
</pre>

助词<span lang="ja">と</span>也用于表达某种"包含"，但它的工作方式与<span lang="ja">も</span>非常不同。在英语翻译层面，它们可能看起来很相似，但不要被迷惑了。它们完全不同。

<span lang="ja">と</span>通常用来列举事物。

<pre>
<span lang="ja">ジム</span><b><span lang="ja">と</span></b><span lang="ja">メアリーは来ない</span>
吉姆<b>和</b>玛丽不会来。
</pre>

<span lang="ja">と</span>还可以用来标记主语与其"一起"做某事的对象。不要把它与<span lang="ja">で</span>混淆，后者可能看起来很相似：区别在于<span lang="ja">と</span>通常用于人（和你一起做某事的人），而<span lang="ja">で</span>用于工具和物品（你用来做某事的工具）。

<pre>
<span lang="ja">私はヤン</span><b><span lang="ja">と</span></b><span lang="ja">勉強する</span>
我<b>和</b>杨一起学习。

<span lang="ja">ペン</span><b><span lang="ja">で</span></b><span lang="ja">書く</span>
<b>用</b>笔写（直译："通过笔"）
</pre>

有时<span lang="ja">で</span>和<span lang="ja">と</span>看起来可以互换，但语气略有不同。比较以下两句：

<pre>
<span lang="ja">みんな</span><b><span lang="ja">と</span></b><span lang="ja">遊ぶ</span>
<b>和</b>大家一起玩

<span lang="ja">みんな</span><b><span lang="ja">で</span></b><span lang="ja">遊ぶ</span>
<b>大家一起</b>玩
</pre>

在英语中，翻译结果实际上是一样的，但第一句中的<span lang="ja">と</span>暗示主语是**和**作为一个额外群体的大家玩（所以是主语 + 其他人）。而在第二句中，主语是群体的一部分，他们是**作为**大家整体一起来玩（"大家"包含主语）。这是一个非常微妙的区别，但了解它是有益的。
