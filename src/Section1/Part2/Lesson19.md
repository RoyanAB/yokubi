# 用<span lang="ja">か</span>提问

日语中疑问句是通过语调来形成的，有时也会在句末加上<span lang="ja">か</span>或<span lang="ja">の</span>，具体取决于礼貌程度。

本课将学习以<span lang="ja">か</span>结尾的是/否疑问句。我们稍后再学习"wh-"疑问句和<span lang="ja">の</span>疑问句。

在书面语中，由于无法传达语调，写作者通常会用问号<span lang="ja">？</span>来表示疑问句的存在，就像英语一样。但需要注意，这并不是必需的，因为疑问句也可以仅以<span lang="ja">か</span>（甚至<span lang="ja">の</span>）和句号<span lang="ja">。</span>结尾，而不使用问号。日语的标点规则与英语不同。

<pre>
<span lang="ja">暑い？</span>
热吗？（普通形）

<span lang="ja">暑いですか。</span>
热吗？（礼貌形）
</pre>

当用<span lang="ja">か</span>构成疑问句时，<span lang="ja">か</span>只需放在句末即可。不过，用<span lang="ja">か</span>提问通常限用于礼貌形（<span lang="ja">ですか？</span>和<span lang="ja">ますか？</span>）。在普通形中，你通常不会用<span lang="ja">か</span>来提问，因为这样听起来可能非常生硬、过于直接。人们通常只是用升调来表示疑问，就像上面的例子那样。

在某些情况下，<span lang="ja">か</span>也可以在普通形中使用，但那通常是反问句，或者带有某种特定的语气，这种语气在初学者指南中很难解释清楚。总之不要掉入那个陷阱，觉得每个疑问句只要在末尾加个<span lang="ja">か</span>就行了，至少在你真正掌握之前不要这样做。

不过，最后这条建议不适用于媒体作品中的虚构语言，这也是现实生活中日语和虚构日语（动漫、漫画、书籍等）之间最常见的差异来源之一。

<pre>
<span lang="ja">お前は警察か</span>
你是警察吗？（听起来很凶，属于虚构语言）
</pre>

如上所示，要在礼貌语中用<span lang="ja">か</span>提问，只需在<span lang="ja">です</span>/<span lang="ja">ます</span>后面加上<span lang="ja">か</span>即可。

<pre>
<span lang="ja">彼は学生です</span><b><span lang="ja">か</span></b><span lang="ja">。</span>
他是学生<b>吗？</b>

<span lang="ja">古いです</span><b><span lang="ja">か</span></b>
它旧<b>吗？</b>

<span lang="ja">食べます</span><b><span lang="ja">か</span></b>
你吃<b>吗？</b>
</pre>

有一些固定短语和表达可能会用<span lang="ja">か</span>来提出看起来像反问句的问题，但它们实际上并不是疑问句。例如：

<pre>
<span lang="ja">あ、そっか。</span>
啊，原来如此。
</pre>

<span lang="ja">そっか</span>是<span lang="ja">そうか</span>的普通形连读，字面意思是"是这样吗？"
