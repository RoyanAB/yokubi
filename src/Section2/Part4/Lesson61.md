# 大约——<span lang="ja">ころ</span>与<span lang="ja">くらい</span>，以及限度——<span lang="ja">まで</span>、<span lang="ja">ほど</span>、<span lang="ja">すぎる</span>

<span lang="ja">ころ</span>是一个大致表示"大约时间"的名词。你会经常遇到X<span lang="ja">のころに</span>。这可以是一个连词"在X的时候左右"，或者是一个带上<span lang="ja">に</span>助词的名词"X发生的大致时间"。在某些情况下，<span lang="ja">ころ</span>可以以后缀<span lang="ja">ごろ</span>的形式直接接在名词后面。

<pre>
<span lang="ja">子供</span><b><span lang="ja">のころに</span></b><span lang="ja">見たアニメを思い出す。</span>
我想起了<b>在小时候</b>看过的动画。

<span lang="ja">そういえば、去年の</span><b><span lang="ja">今ごろ</span></b><span lang="ja">だったかな？</span>
说起来，是<b>去年这个时候左右</b>吧？
</pre>

<span lang="ja">くらい</span>接一个度量或数量（包括时间），使其变为大约。它与<span lang="ja">ぐらい</span>完全可互换，完全取决于说话者偏好。

<pre>
<span lang="ja">午後</span>1<span lang="ja">時</span><b><span lang="ja">ぐらい</span></b><span lang="ja">で大丈夫？</span>
下午1点<b>左右</b>可以吗？
</pre>

它也可以用来表示"到了……的程度"或"到了……的水平"，并且绝不超出那个范围。

<pre>
<span lang="ja">毎日食べたい</span><b><span lang="ja">くらい</span></b><span lang="ja">美味しかったです</span>
好吃<b>到了</b>我想每天都吃<b>的程度</b>。
</pre>

当与<span lang="ja">は</span>助词搭配时，它带有"至少"的细微意味。

<pre>
<span lang="ja">名前</span><b><span lang="ja">くらいは</span></b><span lang="ja">聞いたことがあるだろう。</span>
你<b>至少</b>应该听说过它的名字吧。
</pre>

<span lang="ja">まで</span>基本上表示"直到"或"到……为止"。<span lang="ja">まで</span>所标记的限制或边界是包含在范围内的。

<pre>
<span lang="ja">世界が終わる日</span><b><span lang="ja">まで</span></b><span lang="ja">。</span>
<b>直到</b>世界终结的那一天。
</pre>

<span lang="ja">まで</span>常常与<span lang="ja">から</span>搭配为<span lang="ja">〜から〜まで</span>（从<span lang="ja">〜</span>到<span lang="ja">〜</span>）。

<pre>
<span lang="ja">小さなこと</span><b><span lang="ja">から</span></b><span lang="ja">、大きなこと</span><b><span lang="ja">まで</span></b><span lang="ja">。</span>
<b>从</b>小事<b>到</b>大事。
</pre>

<span lang="ja">までに</span>将<span lang="ja">まで</span>的含义改为"在……之前"（如"在下周五之前完成某事"）。

<pre>
<span lang="ja">明後日</span><b><span lang="ja">までに</span></b><span lang="ja">は終わらせます。</span>
我会<b>在</b>后天<b>之前</b>完成。
</pre>

<span lang="ja">ほど</span>基本上在比较中表示"如同"。在深层含义上，<span lang="ja">ほど</span>的核心意思是"到了……的程度"。

<pre>
<span lang="ja">私は彼ら</span><b><span lang="ja">ほど</span></b><span lang="ja">強くない。</span>
我没有他们<b>那么</b>强。
</pre>

在肯定句中，当<span lang="ja">ほど</span>附加在"数量"上时，它是一个大约值。

<pre>
<span lang="ja">中身はまだ半分</span><b><span lang="ja">ほど</span></b><span lang="ja">残っている。</span>
里面还剩下<b>大约</b>一半的内容物。
</pre>

<span lang="ja">すぎる</span>是一个动词，意为"超过"或"超越"。作为助动词，它表示"太多"或指示一种过度的行为。它附加在<span lang="ja">ます</span>词干上。

<pre>
<span lang="ja">美味しいものを食べ</span><b><span lang="ja">すぎた</span></b><span lang="ja">。</span>
好吃的吃<b>太多了</b>。
</pre>
