# 用<span lang="ja">いい</span>表达许可，用<span lang="ja">ませんか</span>表达否定形式的邀请

<span lang="ja">いい</span>仅仅表示"好"或"可以"，本身并非一个语法术语，但它常用于表达许可的句型中。在<span lang="ja">て</span>形之后，它用来请求允许做<span lang="ja">て</span>所标记的动作。<span lang="ja">てもいい</span>的使用方式也大致相同，两者都很常见。

<pre>
<span lang="ja">食べて</span>(<span lang="ja">も</span>)<b><span lang="ja">いい</span></b><span lang="ja">？</span>
我可以吃吗？
<b><span lang="ja">いい</span></b><span lang="ja">よ</span>
可以呀
</pre>

当人们使用<span lang="ja">よかった</span>时，他们通常是在表达喜悦之情，而不是好或可以。<span lang="ja">よかった</span>不应用于表达许可。

<pre>
<span lang="ja">来てくれて</span><b><span lang="ja">よかった</span></b>
很高兴你来了。
</pre>

当涉及否定时，带<span lang="ja">いい</span>的"许可"陈述常常介于许可和请求之间，就像英语中过度礼貌的表达一样。当涉及否定 + <span lang="ja">ていい</span>时，<span lang="ja">ないで</span>(<span lang="ja">も</span>)<span lang="ja">いい</span>和<span lang="ja">なくて</span>(<span lang="ja">も</span>)<span lang="ja">いい</span>意思大致相同，基本可以互换。<span lang="ja">なくて</span>(<span lang="ja">も</span>)<span lang="ja">いい</span>更常见一些。

<pre>
<span lang="ja">食べなくて</span>(<span lang="ja">も</span>)<b><span lang="ja">いい</span></b><span lang="ja">よ</span>
你不吃也没关系哦
</pre>

作为回应，<span lang="ja">いい</span>(<span lang="ja">です</span>)通常带有拒绝的意思（就像英语中的"I'm good"）。

<pre>
<span lang="ja">レジ袋ご利用ですか</span>?
您需要袋子吗？
<span lang="ja">いや、</span><b><span lang="ja">いいです</span></b>
不，不用了。
</pre>

在前面的课程中，我们看到意志形可以用来邀请/催促某人和你一起做某事。还有另一种方式来问某人是否想和你一起做某事，那就是用否定形式的疑问句。这类似于英语中的"won't you...?"

<pre>
<span lang="ja">明日、映画を見に</span><b><span lang="ja">行かない</span></b><span lang="ja">？</span>
我们明天不去看电影吗？（即：要不要一起去看电影？）
</pre>

这个结构也可用于并非邀请的请求。

<pre>
<span lang="ja">すいません、もう一回言ってくれ</span><b><span lang="ja">ませんか</span></b><span lang="ja">？</span>
不好意思，你能再说一遍吗？（直译："你不愿意再说一遍吗？"）
</pre>
