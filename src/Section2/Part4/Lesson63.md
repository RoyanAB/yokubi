# 再多一点：一些额外的助词和助动词

作为最后一课，让我们谈谈一些无法归入其他任何课程的最后几组助词和助动词。

其中大多数是列举助词或强调助词，它们很难忘记，也很容易掌握。把它们当作临别之际的最后一份知识礼物吧。

<span lang="ja">なり</span>可以是一个列举标记，用于标记可能选项的例子。

<pre>
<span lang="ja">親</span><b><span lang="ja">なり</span></b><span lang="ja">教師</span><b><span lang="ja">なり</span></b><span lang="ja">の指導が必要だ</span>
父母<b>或</b>老师的指导是必要的。
</pre>

<span lang="ja">きり</span>/<span lang="ja">っきり</span>是一个辅助后缀，有三种含义。
- 在 \<名词\><span lang="ja">きり</span> 的模式中，它像一个"仅仅/只有"的标记。
- 在 \<陈述\><span lang="ja">きり</span> 的模式中，它基本上表示"自从\<陈述\>以来一直"。
- 在 \<动词词干\><span lang="ja">きり</span> 的模式中，它可以表示"将\<动词\>做到底"。

<span lang="ja">やら</span>是一个有两种含义的助词。它可以用在疑问句末尾，像<span lang="ja">か</span>一样，但带有不确定和情感强调的细微意味。

<pre>
<span lang="ja">まったく、何を考えているの</span><b><span lang="ja">やら</span></b><span lang="ja">。</span>
真是的，到底在想些什么啊……？
</pre>

它也可以用来列举可能性/例子，就像<span lang="ja">か</span>用于替代选项一样。

<pre>
<span lang="ja">夢の中ならお茶</span><b><span lang="ja">やら</span></b><span lang="ja">お菓子</span><b><span lang="ja">やら</span></b><span lang="ja">出せます。</span>
如果是在梦里的话，我可以端出茶<b>啊</b>点心<b>之类的</b>。
</pre>

<span lang="ja">こそ</span>是一个强调助词，很像英语中强调某个词的方式。换句话说，它直接强调它所接的任何内容。它常常"覆盖"在其他助词如<span lang="ja">が</span>、<span lang="ja">は</span>、<span lang="ja">を</span>等之上，并使它们隐藏。

<pre>
<span lang="ja">こういう時</span><b><span lang="ja">こそ</span></b><span lang="ja">冷静にならないと</span>
<b>正是</b>在这种时候必须保持冷静
</pre>

<span lang="ja">さえ</span>是一个强调助词，根据短语的不同，可以表示"连X都"、"仅X"或"只要X"。

<pre>
<span lang="ja">私</span><b><span lang="ja">さえ</span></b><span lang="ja">いなければ、こんなことにはならなかった。</span>
<b>只要</b>我不在，就不会发生这种事了。

<span lang="ja">これ</span><b><span lang="ja">さえ</span></b><span lang="ja">あればなんとかなるだろう。</span>
<b>只要有</b>这个，总会有办法的吧。
</pre>

<span lang="ja">すら</span>是一个略带文学色彩的强调助词，与<span lang="ja">さえ</span>非常相似。当<span lang="ja">さえ</span>表示"连X都"时，<span lang="ja">すら</span>可以替换<span lang="ja">さえ</span>。

<pre>
<span lang="ja">何が起きたのか</span><b><span lang="ja">すら</span></b><span lang="ja">理解できなかった。</span>
我<b>甚至</b>无法理解到底发生了什么。
</pre>

<span lang="ja">ずつ</span>是一个表示"每个"或"每次"的助词。它强调某事的步调节奏是平稳的。

<pre>
<span lang="ja">ゆっくり、ゆっくりと、一歩</span><b><span lang="ja">ずつ</span></b><span lang="ja">。</span>
慢慢地……慢慢地……一次<b>一步</b>。
</pre>

<span lang="ja">がる</span>是一个助动词，表示说话者以外的某人表现出某种迹象。<span lang="ja">たがる</span>是<span lang="ja">たい</span>和<span lang="ja">がる</span>的组合，表示某人看起来想做某事。

<pre>
<span lang="ja">彼女はそれを知り</span><b><span lang="ja">たがって</span></b><span lang="ja">いた。</span>
她看起来想要知道那个。
</pre>

<span lang="ja">がち</span>是一个助动词，表示某人或某事物有一种倾向。这种倾向从外部视角来看通常是不受欢迎的。

<pre>
<span lang="ja">後ろから、遠慮</span><b><span lang="ja">がち</span></b><span lang="ja">に声が掛けられた。</span>
从身后传来了一声<b>带着</b><b>几分</b>犹豫的招呼。
</pre>
