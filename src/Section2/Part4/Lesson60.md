# 预期与应该——<span lang="ja">わけ</span>、<span lang="ja">はず</span>、<span lang="ja">べき</span>、<span lang="ja">ものだ</span>与<span lang="ja">かもしれない</span>

<span lang="ja">わけ</span>是一个表示"理由"（即"为什么"）的名词。它可以用于许多不同的固定表达。这里无法列出全部，仅举几个常见的。

核心上，<span lang="ja">わけ</span>(<span lang="ja">だ</span>)陈述一个理由、解释、事实、逻辑重述或结论。这种用法有时可以替换为<span lang="ja">のだ</span>。

<pre>
<span lang="ja">なるほど、そういう可能性もある</span><b><span lang="ja">わけだ</span></b><span lang="ja">。</span>
原来如此，<b>所以</b>那种可能性也是存在的<b>啊</b>。

<span lang="ja">それならいくら探しても見つからない</span><b><span lang="ja">わけだ</span></b><span lang="ja">。</span>
如果那样的话，<b>难怪</b>怎么找也找不到她。
</pre>

<span lang="ja">わけがない</span>表示"不可能"，暗示某事在说话者看来甚至完全无法想象。

<pre>
<span lang="ja">そんな馬鹿なことがある</span><b><span lang="ja">わけがない</span></b><span lang="ja">。</span>
<b>不可能</b>有那种荒唐的事。
</pre>

另一方面，<span lang="ja">わけじゃない</span>是一种委婉地否定某种情况、形势或逻辑结论的方式。

<pre>
<span lang="ja">可能性がない</span><b><span lang="ja">わけじゃない</span></b><span lang="ja">。</span>
<b>也不是说</b>没有可能。
</pre>

<span lang="ja">わけにはいかない</span>表示根据常识或社会预期某事是不可能的。

<pre>
<span lang="ja">このチャンスを逃す</span><b><span lang="ja">わけにはいかない</span></b><span lang="ja">。</span>
<b>我不能</b>放过这个机会。
</pre>

由于<span lang="ja">わけ</span>是一个非常强烈的断言，它常常与<span lang="ja">という</span>搭配，以<span lang="ja">というわけ</span>的形式来软化表达的影响力。

<span lang="ja">はず</span>表示说话者预期某事在事实上为真或会成真。它可能类似于英语中的"should"，但覆盖的情况范围并不完全相同。<span lang="ja">はず</span>也暗示这种预期是基于推理的，而不是纯粹的猜测。

<pre>
<span lang="ja">こんな</span><b><span lang="ja">はず</span></b><span lang="ja">じゃなかった、こんな</span><b><span lang="ja">はず</span></b><span lang="ja">じゃなかった</span>
<b>不应该</b>是这样的，<b>不应该</b>是这样的

<span lang="ja">さっきまで自分の部屋にいた</span><b><span lang="ja">はず</span></b><span lang="ja">なのに。</span>
可是她<b>应该</b>直到刚才还在自己房间里的……

<span lang="ja">その場にいた全員がそう思った</span><b><span lang="ja">はず</span></b><span lang="ja">だ。</span>
在场的所有人<b>肯定</b>都那么想了。
</pre>

<span lang="ja">べき</span>表示说话者预期某种行为或状态应被遵守，因为不这样做会是不好的、不负责任的等。

<pre>
<span lang="ja">この場から立ち去る</span><b><span lang="ja">べきだ</span></b><span lang="ja">。</span>
<b>最好</b>离开这个地方。

<span lang="ja">わからないことは聞く</span><b><span lang="ja">べきだ</span></b><span lang="ja">。</span>
不懂的事情<b>应该</b>去问。
</pre>

然而，在某些情况下使用<span lang="ja">べき</span>并不总是那么沉重，尤其是在提问寻求他人智慧或建议的时候。

<pre>
<span lang="ja">どこに行く</span><b><span lang="ja">べき</span></b><span lang="ja">だと思う？</span>
你觉得我们<b>应该</b>去哪儿？
</pre>

当处于过去式如<span lang="ja">べきだった</span>时，它表示某个行为或状态"曾经是"或"本来会是"正确的，通常是后者。

<pre>
<span lang="ja">もっと気を付ける</span><b><span lang="ja">べきだった</span></b><span lang="ja">。</span>
我们<b>本该</b>更加小心的。
</pre>

<span lang="ja">べし</span>是<span lang="ja">べき</span>的古语版本。只需知道它的存在即可。

<span lang="ja">ものだ</span>也可以表达"应该"的含义，带有一种普遍的"应该"的细微意味，适用于很多人，而不仅仅是某个特定的人或群体。它常用于陈述"事物本来的样子"。

<pre>
<span lang="ja">子供は親の言う事を聞く</span><b><span lang="ja">ものだ</span></b><span lang="ja">。</span>
孩子<b>就应该</b>听父母的话。
</pre>

<span lang="ja">かもしれない</span>是一个复合句末表达。它的连接方式与单独的<span lang="ja">か</span>相同。它表示说话者认为该陈述是一种可能性。有时人们会出于礼貌，把它加到他们知道是真的东西上。

<pre>
<span lang="ja">まだ寝ぼけている</span><b><span lang="ja">かもしれない</span></b><span lang="ja">。</span>
她<b>也许</b>还半梦半醒的。
</pre>

<span lang="ja">かもしれない</span>在日常对话中常常被简化为<span lang="ja">かも</span>。
