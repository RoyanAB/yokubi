# 用<span lang="ja">てみる</span>尝试做某事

<span lang="ja">てみる</span>助动词源于动词<span lang="ja">見る</span>，这有助于记住它的含义。它大致表示"尝试做某事（看看结果如何）"。然而，当用此含义时，它不应写成汉字。

<span lang="ja">てみる</span>表示动作是"尝试"进行的，或者做这个动作的目的是想看看结果会怎样。

<pre>
<span lang="ja">食べ</span><b><span lang="ja">てみる</span></b>
我试着吃吃看（看看结果如何/看看我是否喜欢/看看有没有毒等）

<span lang="ja">チョコレートケーキを作っ</span><b><span lang="ja">てみました</span></b>
我试着做了巧克力蛋糕。

<span lang="ja">想像し</span><b><span lang="ja">てみて</span></b><span lang="ja">ください。</span>
请试着想象一下。
</pre>

在上一课中，我们看到了另一个也可以翻译为"试图做"的语法点：<span lang="ja">ようとする</span>。

在日语中，这两个语法点非常不同，但翻译成英语时常常引起混淆。<span lang="ja">てみる</span>假定动作会被完成，重要的是结果，而<span lang="ja">ようとする</span>暗示困难的部分（"尝试"）是动作本身。它通常带有主语首先就根本无法"做到"该动作的假设。

考虑以下两个句子的区别：

<pre>
<span lang="ja">まずは一度逃げ</span><b><span lang="ja">てみよう</span></b>
首先，我们<b>试着</b>逃一次吧。

<span lang="ja">僕は、逃げ</span><b><span lang="ja">ようとする</span></b><span lang="ja">彼女の手を取った。</span>
我抓住了那位正在<b>试图</b>逃跑的女孩的手。
</pre>

第一句侧重于说话者不知道逃跑尝试会如何发展，但想试试看。第二句则是在女孩甚至还没开始逃跑之前就阻止了她。它暗示她<b>试图</b>启动"逃跑"这个行为，但没能完成。
