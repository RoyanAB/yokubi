# 用<span lang="ja">と思う</span>和<span lang="ja">と考える</span>表达思考和感觉

动词<span lang="ja">思う</span>和<span lang="ja">考える</span>通常都被翻译为"思考"。然而，<span lang="ja">考える</span>涉及一种更条理化/分析性的思维方式，而<span lang="ja">思う</span>则更侧重于那些浮现于脑海中的（有时是无意识的）思想感受。你可以把<span lang="ja">考える</span>理解为"考虑"或"评估"，虽然并不总是那么严肃。

我们已经在[第30课](../Part3/Lesson30.md)中看到<span lang="ja">と</span>助词用于引述想法。如果忘了，请复习那节课。<span lang="ja">考える</span>的使用方式相同。

<pre>
<span lang="ja">何かトラブルがあった</span><b><span lang="ja">と考える</span></b><span lang="ja">のが普通だろう。</span>
我觉得<b>考虑</b>到可能发生了什么麻烦是很正常的吧。
</pre>

回到<span lang="ja">と思う</span>，其基本用法是表达简单的想法或感觉。它通常表示**说话者**本人认为某事。

<pre>
<span lang="ja">来た</span><b><span lang="ja">と思った</span></b> 
<b>我以为</b>他来了。
</pre>

这种思考通常是自然而然的、未经计划的。

动词<span lang="ja">思う</span>的一个特点是，当我们谈论**他人**时，该动词通常改用<span lang="ja">ている</span>形。在这种用法中，它可以暗示我们有一些理由来假定他人的想法（也许他们曾口头表达过自己的意见）。

<pre>
<span lang="ja">彼は本気でそれが正しい</span><b><span lang="ja">と思っている</span></b><span lang="ja">のだ。</span>
他真心地<b>认为</b>那是对的。
</pre>

最后，当<span lang="ja">と思う</span>与意志形短语连用时，表示说话者正在考虑做某事。在这种用法中，它类似于<span lang="ja">たい</span>，但更具体，侧重于实际的、相对短期的计划动作，而非一种欲望状态。

<pre>
<span lang="ja">食べ</span><b><span lang="ja">ようと思う</span></b>
我想我要吃
</pre>
