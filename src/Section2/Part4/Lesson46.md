# "仅仅/只有"——<span lang="ja">だけ</span>、<span lang="ja">のみ</span>、<span lang="ja">ばかり</span>与<span lang="ja">しか</span>

<span lang="ja">だけ</span>、<span lang="ja">のみ</span>和<span lang="ja">ばかり</span>是"仅仅"或"只有"的变体。<span lang="ja">しか</span>类似，但只能用于否定句中。

<span lang="ja">だけ</span>在数量方面表示"恰恰就是"。它像助词一样作用。

<pre>
<span lang="ja">俺</span><b><span lang="ja">だけ</span></b><span lang="ja">行く</span>
只有我去

<span lang="ja">俺</span><b><span lang="ja">だけ</span></b><span lang="ja">じゃないよ</span>
不止是我哦
</pre>

<span lang="ja">のみ</span>就像不那么口语化的<span lang="ja">だけ</span>。

<pre>
<span lang="ja">味付けは塩コショウ</span><b><span lang="ja">のみ</span></b><span lang="ja">。</span>
调味料只有盐和胡椒
</pre>

<span lang="ja">ばかり</span>是另一个"只有"的词。它也可以念作<span lang="ja">ばっか</span>或<span lang="ja">ばっかり</span>。X<span lang="ja">ばかり</span>关注的是所有事物都落在X的定义范围内为真，而<span lang="ja">だけ</span>和<span lang="ja">のみ</span>关注的是超出该陈述范围的事物为假。想想"只有这个"和"除此之外没有别的"之间语感的差异。

<pre>
<span lang="ja">悪いこと</span><b><span lang="ja">ばかり</span></b><span lang="ja">じゃないと思います。</span>
我认为并不全是坏事。

<span lang="ja">分からないこと</span><b><span lang="ja">ばかり</span></b><span lang="ja">だった</span>
全都是我不懂的事情。
</pre>

结构\<动词\><span lang="ja">てばかりいる</span>常用以传达一种批评的态度，指责某人只做一件事情反反复复，到了有害的程度。

<pre>
<span lang="ja">彼は寝て</span><b><span lang="ja">ばかり</span></b><span lang="ja">いる</span>
他除了睡觉什么都不做
</pre>

<span lang="ja">ばかり</span>也可与时间性陈述一起使用，传达某事发生得有多快或多早。

<pre>
<span lang="ja">戦いはまだ始まった</span><b><span lang="ja">ばかり</span></b><span lang="ja">だ。</span>
战斗才刚刚开始。
</pre>

<span lang="ja">しか</span>与否定句一起使用，传达与<span lang="ja">だけ</span>相同的含义，但侧重于除被标记的事物之外的所有其他事物。

<pre>
<span lang="ja">僕</span><b><span lang="ja">しか</span></b><span lang="ja">いない</span>
除了我以外没有别人在这里
</pre>

复合词<span lang="ja">だけしか</span>也存在。它与<span lang="ja">しか</span>的用法完全一样，但具有更强的强调效果。
