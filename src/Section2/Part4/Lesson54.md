# 用<span lang="ja">なる</span>和<span lang="ja">する</span>表达变成和使成为

我们已经在之前课程的一些例子中看到过<span lang="ja">なる</span>。现在让我们更深入地看看它的用法。

<span lang="ja">なる</span>是一个动词，含义类似"变成"或"最终成为"。除了单单"变成"之外，它还可以用于各种间接状态，比如一个团队最终由某些成员组成，或者谣言变成真的。

<pre>
<span lang="ja">海賊王</span><b><span lang="ja">に</span></b><span lang="ja">俺は</span><b><span lang="ja">なる</span></b><span lang="ja">！</span>
我要<b>成为</b>海贼王！

1500<span lang="ja">円</span><b><span lang="ja">になります</span></b>
是1500日元。
</pre>

<span lang="ja">に</span>被用作标记主语"变成"的名词（或事物）的助词。与往常一样，助词的顺序可以改变强调的重点，正如我们在上面第一个例句中看到的那样。

然而实际上，<span lang="ja">なる</span>实际上接副词，而不仅仅是带有<span lang="ja">に</span>标记的名词。只是恰好在使用<span lang="ja">に</span>时，名词（和<span lang="ja">な</span>形容词）可以变成副词形式。

当我们尝试将<span lang="ja">なる</span>与<span lang="ja">い</span>形容词一起使用时就可以确认这一点。为此，我们需要这样将它们变成副词：

<pre>
<span lang="ja">これからもっともっと強</span><b><span lang="ja">くなる</span></b><span lang="ja">。</span>
从现在开始，我要<b>变得</b>越来越强。
</pre>

否定（<span lang="ja">ない</span>）形也一样：

<pre>
<span lang="ja">食事が美味しく</span><b><span lang="ja">なくなる</span></b>
饭<b>变得</b>不好吃了
</pre>

甚至动词也可以与<span lang="ja">なる</span>搭配使用。否定形的动词像<span lang="ja">い</span>形容词一样活用，<span lang="ja">たい</span>形的动词也是如此。而普通/非否定形的动词则需要一个额外的辅助名词如<span lang="ja">よう</span>，它大致表示"以……的方式"（我们在[第42课](../Part3/Lesson42.md)中已经见过的同一个<span lang="ja">よう</span>）。

<pre>
<span lang="ja">たこ焼き食べた</span><b><span lang="ja">くなった</span></b><span lang="ja">。</span>
我<b>变得</b>想吃章鱼烧了。

<span lang="ja">ここで下がれば本当に道がわから</span><b><span lang="ja">なくなる</span></b><span lang="ja">。</span>
如果从这里往下走，就真的会<b>变得</b>找不到路了。

<span lang="ja">見えなかったものが見える</span><b><span lang="ja">ようになる</span></b><span lang="ja">。</span>
原本看不见的东西<b>变得</b>看得见了。
</pre>

<span lang="ja">ようになる</span>这个结构常常将动词变为可能形，表示"变得能够做"某事。

作为\<副词\> + <span lang="ja">なる</span>的对应结构，我们现在来看看\<副词\> + <span lang="ja">する</span>。

你可以将<span lang="ja">する</span>视为<span lang="ja">なる</span>的"主动"版本。\<副词\> + <span lang="ja">なる</span>表示"**变成**\<副词\>"，而\<副词\> + <span lang="ja">する</span>表示"**使**（某事物）变成\<副词\>"。

句子结构与<span lang="ja">なる</span>完全相同，只是我们用<span lang="ja">を</span>来标记被"使成为"的事物。

<pre>
<span lang="ja">みんなが目を丸</span><b><span lang="ja">くする</span></b><span lang="ja">。</span>
大家都瞪大了眼睛（直译："把眼睛变成了圆形"）

<span lang="ja">彼女は笑みを深</span><b><span lang="ja">くする</span></b><span lang="ja">。</span>
她加深了微笑（直译："让微笑变深"）

<span lang="ja">美味しいものは人を幸せ</span><b><span lang="ja">にする</span></b><span lang="ja">。</span>
好吃的东西让人幸福（直译："把人变成幸福"）
</pre>
