# 更多否定形式——<span lang="ja">なくて</span>、<span lang="ja">ないで</span>、<span lang="ja">ず</span>与<span lang="ja">ずに</span>

我们在[第11课](../../Section1/Part1/Lesson11.md)中简要讨论过<span lang="ja">なくて</span>和<span lang="ja">ないで</span>的区别。让我们再深入挖掘一下。

当我们将动词或<span lang="ja">い</span>形容词变成<span lang="ja">なくて</span>形时，其表现就像通常的<span lang="ja">て</span>连接形式（"做X，然后……"）。在这种用法中，常常暗示某种因果序列（"我没有做X，这就是为什么Y发生了"），虽然并非严格要求如此。

<pre>
<span lang="ja">お役に立て</span><b><span lang="ja">なくて</span></b><span lang="ja">すみません。</span>  
对不起我<b>没能</b>帮上忙（直译："我帮不上忙<b>所以</b>对不起"）
</pre>

我们也可以用<span lang="ja">ないで</span>来构成否定，然而与<span lang="ja">なくて</span>相对比，这种用法常常暗示后续动作进行的方式或状态。你可以把<span lang="ja">ないで</span>之前的陈述看作是对其后陈述的"副词"。

<pre>
<span lang="ja">もしかして、食べ</span><b><span lang="ja">ないで</span></b><span lang="ja">待っていたのか</span>
也许……你<b>没吃东西就</b>等着吗？

<span lang="ja">ノックもし</span><b><span lang="ja">ないで</span></b><span lang="ja">部屋に入る。</span>
<b>连门都不敲就</b>进房间。
</pre>

还有另一种使用<span lang="ja">なくて</span>和<span lang="ja">ないで</span>的方式，那就是使用动词形式<span lang="ja">〜ず</span>和<span lang="ja">〜ずに</span>。<span lang="ja">ず</span>是否定助动词<span lang="ja">ぬ</span>的古语活用形式，但你真的不需要知道这个。只需记住，<span lang="ja">〜ず</span>几乎是与<span lang="ja">なくて</span>（或在某些情况下<span lang="ja">なく</span>）一一对应的替换，而<span lang="ja">〜ずに</span>是<span lang="ja">ないで</span>的替换。在**某些**情况下，单独的<span lang="ja">〜ず</span>（不带<span lang="ja">に</span>）也可以像<span lang="ja">ないで</span>一样使用（耶，日语就是这样！）。

<pre>
<span lang="ja">お役に立て</span><b><span lang="ja">ず</span></b><span lang="ja">、申し訳ございません。</span>
对不起我<b>没能</b>帮上忙

<span lang="ja">何も食べ</span><b><span lang="ja">ずに</span></b><span lang="ja">眠ってしまった。</span>
他什么都<b>没</b>吃就睡着了
</pre>

<span lang="ja">ず</span>和<span lang="ja">ずに</span>比它们的<span lang="ja">なくて</span>和<span lang="ja">ないで</span>对应形式更正式、更生硬，大多用于书面语。然而，你在口语中仍然会听到不少<span lang="ja">ずに</span>，并且它们在常见词组和表达中出现得很频繁。
