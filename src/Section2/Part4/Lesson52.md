# 更多疑问形式——<span lang="ja">っけ</span>、<span lang="ja">かな</span>、<span lang="ja">かい</span>、<span lang="ja">だい</span>、<span lang="ja">じゃない</span>与<span lang="ja">じゃん</span>

<span lang="ja">っけ</span>用于说话者忘记了某事或试图回忆起曾经知道的事情时。它通常（尽管并非总是）用作自言自语的问题。

<pre>
<span lang="ja">なんだ</span><b><span lang="ja">っけ</span></b><span lang="ja">？</span>
是什么来着？

<span lang="ja">どこに置いた</span><b><span lang="ja">っけ</span></b>
我把东西放哪儿了来着？
</pre>

<span lang="ja">っけ</span>几乎从不与非名词一起使用，除非是过去式。当用于非名词时，通常会在中间插入其他内容（比如解释性的<span lang="ja">んだ</span>）。

<pre>
<span lang="ja">どこにあった</span><b><span lang="ja">っけ</span></b><span lang="ja">？</span>
它在哪儿来着？

<span lang="ja">こういうの何て言うんだ</span><b><span lang="ja">っけ</span></b><span lang="ja">。</span>
这个东西叫什么来着？
</pre>

与<span lang="ja">っけ</span>类似，还有<span lang="ja">かな</span>。它通常以升调发音作为疑问，但就像<span lang="ja">っけ</span>一样，也是一个自我导向的问题。它简单表示说话者正在疑惑、不确定或犹豫某事，类似于英语中的"I wonder"。不用说，它来自疑问助词<span lang="ja">か</span>和句末助词<span lang="ja">な</span>的组合。

<pre>
<span lang="ja">話したいことがあるんだけど、いい</span><b><span lang="ja">かな</span></b><span lang="ja">？</span>
我有事情想和你说，不知道现在方便吗？
</pre>

它常常出现在动词的意志形之后，表示说话者即将决定做某事。

<pre>
<span lang="ja">さて、本でも読もう</span><b><span lang="ja">かな</span></b>
嗯，要不读本书什么的吧？
</pre>

<span lang="ja">かい</span>是一个"友好"的疑问标记，尤其适用于男性说话者。在熟悉的对话之外使用，可能会显得粗鲁/过于直接。它在现实生活中不如在媒体作品中常见。在标准用法中，它只能用来问是非问题。

<pre>
<span lang="ja">お嬢ちゃん、大丈夫</span><b><span lang="ja">かい</span></b><span lang="ja">？</span>
小姑娘，你还好吗？
</pre>

与<span lang="ja">かい</span>并行的是<span lang="ja">だい</span>。其语感大致相同，但与<span lang="ja">かい</span>不同，<span lang="ja">だい</span>用于询问期望得到实际回答的问题。

<pre>
<span lang="ja">君たちはこれからどうするん</span><b><span lang="ja">だい</span></b><span lang="ja">？</span>
你们今后打算怎么办？
</pre>

<span lang="ja">じゃない</span>也可以用来询问确认性问题。是的，当接在名词后时，它与普通的"不是"完全一样。不，没办法绕开这点。如果你运气好，后面会有一个问号。

<pre>
<span lang="ja">いい</span><b><span lang="ja">じゃない</span></b>
挺好的，不是吗？
</pre>

请注意，在动词和<span lang="ja">い</span>形容词后面加断定助词在技术上是错误的，但这类<span lang="ja">じゃない</span>有自己独立的存在方式，可以出现在它们之后。它并非否定谓语，只是在添加一种否定疑问的细微含义（"不是吗？"），并且表现得像一个句末助词。

<span lang="ja">じゃん</span>是<span lang="ja">じゃない</span>的口语缩约形式。
