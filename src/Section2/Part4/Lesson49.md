# <span lang="ja">とする</span>的多种用法

<span lang="ja">とする</span>有四种主要用法。第一种是表示某人假定/假设某事，与陈述句一起使用。可以将其理解为"假装……/表现得好像……"。这种方式最容易被视为一种邀请，但也可以是关于他人假定的较长句子的一部分。

<pre>
<span lang="ja">それが本当だ</span><b><span lang="ja">とすれば</span></b><span lang="ja">どうなるのだろうか。</span>
<b>假设</b>那是真的，我想知道会怎么样呢？

<span lang="ja">誰かが呼びに来た</span><b><span lang="ja">としても</span></b><span lang="ja">不思議ではない。</span>
<b>即使假设</b>有人来这里叫（我们？），也不奇怪。
</pre>

第二种是作为<span lang="ja">として</span>助词使用，具有一般的模糊含义"作为X运作"。这与第一种用法直接相关，可以把它理解为"将Y视为X"。在这种用法中，它通常直接接在名词之后。

<pre>
<span lang="ja">その判断は医者</span><b><span lang="ja">として</span></b><span lang="ja">間違いではなかった。</span>
那个判断，<b>作为一名</b>医生，并不是错误的。
</pre>

第三种用法表示某事物经历了一种"体验性"的状态，基本上与像<span lang="ja">つるつる</span>"光滑"这样的"拟态"词一起使用。拟态词是指代表声音（拟声词）或类似声音的属性的词。

<pre>
<span lang="ja">頬がぴりっ</span><b><span lang="ja">とする</span></b><span lang="ja">。</span>
我感到脸颊一阵刺痛。
</pre>

最后，<span lang="ja">とする</span>可以出现在意志形动词之后。这可以赋予它两种不同但通常相似的含义。

第一种含义是某事"即将发生"。既可以是在主语控制下的意志性动作，也可以是某种不由自主和不可避免的（如自然行为等）。

<pre>
<span lang="ja">本部を出て行</span><b><span lang="ja">こうとした</span></b><span lang="ja">時、アルフォンスに呼ばれた。</span>
当我<b>正要</b>离开总部时，被阿尔冯斯叫住了。

<span lang="ja">そんな時間も終わりを迎え</span><b><span lang="ja">ようとしている</span></b><span lang="ja">。</span>
那段时光也<b>即将</b>迎来终结。
</pre>

另一种含义更接近"试图做某事"，只能用于意志性动作。

<pre>
<span lang="ja">本気で僕を食べ</span><b><span lang="ja">ようとしている</span></b><span lang="ja">らしい。</span>
看来它好像真的在<b>试图</b>吃掉我。
</pre>

当动词是意志性动作时，有时"即将"和"试图"含义之间的界限可能是模糊的。想一想，如果你试图做某事但因为被打断而失败，你当时也正处于即将做的状态。因此这个结构可以同时兼有两种含义：

<pre>
<span lang="ja">俺が再びご飯を食べ</span><b><span lang="ja">ようとした</span></b><span lang="ja">時、ガタッと音がする。</span>
当<b>我试图</b>再次吃饭时，我听到了嘎吱一声。
当<b>我正要</b>再次吃饭时，我听到了嘎吱一声。
</pre>
