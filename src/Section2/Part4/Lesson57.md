# 遗憾与心安的助动词——<span lang="ja">てしまう</span>与<span lang="ja">ておく</span>

<span lang="ja">てしまう</span>表示某事"完成"或"彻底做完"。它也用于事物"最终"进入如此"完成"的状态，比如犯错之后。记住它的一个好方法是想到"事情做完了"和"这下可好了（闯祸了）"。它常常带有一种遗憾的细微含义。

<span lang="ja">てしまう</span>在口语中也可以是<span lang="ja">ちまう</span>或<span lang="ja">ちゃう</span>（对于浊音动词活用形式，如<span lang="ja">死んでしまう</span> \-\> <span lang="ja">死んじゃう</span>）。

<pre>
<span lang="ja">私はあっという間に全部食べ</span><b><span lang="ja">てしまった</span></b><span lang="ja">。</span>
我一眨眼的功夫就<b>把</b>全部都吃<b>完了</b>。

<span lang="ja">私は死ん</span><b><span lang="ja">でしまった</span></b><span lang="ja">みたいです。</span>  
我好像是死<b>了</b>（直译：<b>最终落得</b>死去的状态）
</pre>

一旦某事用<span lang="ja">てしまう</span>完成，就意味着它无法挽回。

<span lang="ja">ておく</span>表示做某事，然后不管它/放在一边，并期望它将来会结出果实。这通常被描述为"预先做好准备"，因为实际上这就是它常常暗示的含义。它也可以用作比喻，表示"做了某事然后把它抛之脑后"。<span lang="ja">ておく</span>常缩约为<span lang="ja">とく</span>。

<pre>
<span lang="ja">まず最初に言っ</span><b><span lang="ja">ておき</span></b><span lang="ja">たいことがある。</span>
首先，有件事我<b>想先</b>跟你说<b>清楚</b>。

<span lang="ja">俺はただ知って</span><b><span lang="ja">おきたい</span></b><span lang="ja">だけなのだ。</span>
我只是单纯想<b>知道</b>而已（= 为了让自己安心）
</pre>
