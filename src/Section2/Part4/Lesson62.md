# 时间参照——<span lang="ja">後</span>、<span lang="ja">前</span>、<span lang="ja">先</span>与<span lang="ja">時</span>

<span lang="ja">後</span>是一个表示"之后"的名词。X<span lang="ja">後で</span>是一个连词，表示某事在X之后发生，尽管不一定是紧接着。

<pre>
<span lang="ja">昨日は夕食を食べた</span><b><span lang="ja">あとで</span></b><span lang="ja">、すぐに部屋に戻った。</span>
昨天，<b>在</b>吃完晚饭<b>之后</b>，我们立刻回到了自己的房间。
</pre>

<span lang="ja">前</span>是<span lang="ja">後</span>的反义词，意为"之前"。<span lang="ja">前</span>既可以表示空间上"前面"（即在你前面），也可以表示时间上"过去/以前"。这似乎有些反直觉，因为人们常说"把过去抛在身后"，但如果你想象一列排队的人，排在你*前面*的人会*先于*你轮到自己。所以当轮到你的时候，他们的已经在*过去*了。

连词X<span lang="ja">前に</span>表示某事在X之前发生，不一定是紧接着。如果X是一个动词，它**不能**使用过去式。

<pre>
<span lang="ja">考える</span><b><span lang="ja">前に</span></b><span lang="ja">、体が動いていた。</span>
<b>在</b>思考<b>之前</b>，身体就已经动了起来。
</pre>

<span lang="ja">前</span>和<span lang="ja">先</span>几乎是同义词。

<span lang="ja">先</span>是一个名词，意为"之前"、"前头"或"前端/末端"。一般来说，当<span lang="ja">先</span>是空间性的时，它指的是某事物的前端（"尖端"），或者它所朝向的方向，而不是它正前方的位置。

<pre>
<span lang="ja">目線の</span><b><span lang="ja">先に</span></b><span lang="ja">は、少女がいた。</span>
在视线<b>的前方</b>，有一个少女。（直译："在我视线的延伸方向上"）
</pre>

就像<span lang="ja">前</span>一样，它在时间上也可以表示"之前"。

<pre>
<span lang="ja">それは</span><b><span lang="ja">先に</span></b><span lang="ja">言った。</span>
那个我<b>之前</b>说过了。
</pre>

<span lang="ja">さっき</span>是<span lang="ja">先</span>的口语版本，专门用来表示"片刻之前"。

<span lang="ja">時</span>是一个表示"时间"的名词，但如果你给它一个关系从句，它就可以像连词一样作用（"当……的时候"或"在……之时"）。

<pre>
<span lang="ja">そんなことを考えている</span><b><span lang="ja">時、</span></b><span lang="ja">宿屋の入り口で大声が聞こえた。</span>  
<b>当</b>我正在想着那些事<b>的时候</b>，听到旅馆入口处传来一声大喊。
</pre>

动词 + <span lang="ja">時</span>的行为表现取决于其前后语句中动词的时态。有各种规则，但一般来说，如果<span lang="ja">時</span>之前的动词是过去式，那么它表示<span lang="ja">時</span>之后的动作发生在其之前的动作已经完成之后。不过不要试图死记这些规则，在前进的过程中学会识别它们即可。

<pre>
<span lang="ja">目が覚め</span><b><span lang="ja">た時</span></b><span lang="ja">、俺は知らない場所にいた。</span>
<b>当</b>我醒来<b>的时候</b>，我发现自己在一个陌生的地方。
</pre>
