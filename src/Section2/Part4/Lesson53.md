# 使役形

<span lang="ja">せる</span>是一个"使役"助动词形。它表示使或让某人做某事的动作。是的，它同时用于**强迫**他人和**允许**他人。

对于五段动词，<span lang="ja">せる</span>接在否定词干上。对于一段动词，在<span lang="ja">せる</span>和词干之间插入<span lang="ja">さ</span>：

<pre>
<span lang="ja">終わら</span><b><span lang="ja">せる</span></b>
使（某事物）结束

<span lang="ja">死な</span><b><span lang="ja">せる</span></b>
让（某人）死

<span lang="ja">食べ</span><b><span lang="ja">させる</span></b>
让（某人）吃
</pre>

<div class="warning">
现实提醒：存在例外和简短/口语形式（<span lang="ja">書かせる</span> -> <span lang="ja">書かす</span>）。你不需要太担心它们，但要知道它们的存在。
</div>

当我们在使役形中使用动词时，施事者/动因者（= 执行"强迫"或"允许"的一方）通常由<span lang="ja">は</span>或<span lang="ja">が</span>标记。受事者/目标（= 被"强迫"或"允许"的一方）可以用<span lang="ja">に</span>或<span lang="ja">を</span>标记。这两个助词之间的选择取决于所使用的动词。

当我们使用<span lang="ja">に</span>时，使役的解释偏向"允许"而非"强迫"。另一方面，当我们使用<span lang="ja">を</span>时，两者都有可能。

对于自动词，目标可以用<span lang="ja">を</span>标记，但对于他动词，目标只能用<span lang="ja">に</span>标记。在这种用法中，对于他动词，有时很难判断句子是"允许"还是"强迫"的意思。

<pre>
<span lang="ja">子供</span><b><span lang="ja">を</span></b><span lang="ja">外で遊</span><b><span lang="ja">ばせる</span></b><span lang="ja">。</span>
让孩子们在外面玩。

<span lang="ja">子供</span><b><span lang="ja">に</span></b><span lang="ja">飲</span><b><span lang="ja">ませる</span></b><span lang="ja">酒はない。</span>
我不会让/使孩子喝酒。（直译："没有我会让/使孩子喝的酒。"）
</pre>

使役形也可以与被动态连用（<span lang="ja">せる</span> \-\> <span lang="ja">せられる</span>）。

<pre>
<span lang="ja">書く</span> 写
<span lang="ja">書かせる</span> 让（某人）写
<span lang="ja">書かせられる</span> 被（某人）让写
</pre>

使役被动句不能表示"被允许"的含义，只能表示"被强迫"的含义。

当与<span lang="ja">する</span>动词一起使用时，使役形可以将常用于自动词意义的<span lang="ja">する</span>动词转变为具有明确他动词意义（<span lang="ja">する</span> -> <span lang="ja">させる</span>）。

<pre>
<span lang="ja">空気が乾燥</span><b><span lang="ja">している</span></b><span lang="ja">。</span>
空气（正）干燥。（自动词）

<span lang="ja">果物を乾燥</span><b><span lang="ja">させた</span></b><span lang="ja">お菓子。</span>
用被干燥处理过的水果做的点心。（他动词）
</pre>
