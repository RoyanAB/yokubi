# 用<span lang="ja">ていく</span>和<span lang="ja">てくる</span>表达方向性

我们在[第11课](../../Section1/Part1/Lesson11.md)中已经通过给予助动词<span lang="ja">てあげる</span>、<span lang="ja">てもらう</span>和<span lang="ja">てくれる</span>简要接触了方向性的概念。现在让我们更深入地探讨这个概念，并介绍另外两个助动词：<span lang="ja">ていく</span>和<span lang="ja">てくる</span>。

当不作为助动词使用时，V<span lang="ja">ていく</span>和V<span lang="ja">てくる</span>就是它们字面的意思："做V，然后去/来某处"。然而，当作为助动词使用时，它们的含义会深化，变得比各自部分的总和更丰富。

助动词<span lang="ja">ていく</span>大致表示某个事件正在进行，并将朝着远离说话者的方向"持续下去"，无论是在空间上还是时间上（现在 \-\> 未来）。<span lang="ja">ていく</span>可以缩约为<span lang="ja">てく</span>。

<pre>
<span lang="ja">それはどんどんと大きくなっ</span><b><span lang="ja">ていく</span></b><span lang="ja">。</span>
那东西在逐渐<b>变得</b>越来越<b>大</b>
</pre>

类似地，助动词<span lang="ja">てくる</span>表示某个事件一直持续，从远方朝着说话者靠近，无论是在空间上还是时间上（过去 \-\> 现在），甚至可能进一步继续。

<pre>
<span lang="ja">声が背後から聞こえ</span><b><span lang="ja">てくる</span></b><span lang="ja">。</span>
我听到一个声音从背后<b>传来</b>。
</pre>

<span lang="ja">ていく</span>和<span lang="ja">てくる</span>在用法上都有着极其微妙的差异，本课甚至还没有开始触及这种复杂性的表面，尤其是当你将不同时态混合进来的时候。

我们先看一些例子，然后继续前进。你会在更多的实践中逐渐领会到一些细微差别和常见表达方式。

<pre>
<span lang="ja">これまで見えなかったものが見えるようになっ</span><b><span lang="ja">てきた</span></b><span lang="ja">。</span>
直到现在为止还看不见的东西，已经开始变得看得见了。

<span lang="ja">周囲が騒がしくなっ</span><b><span lang="ja">てきた</span></b><span lang="ja">。</span>
周围开始变得嘈杂起来。

<span lang="ja">顔色がだんだん悪くなっ</span><b><span lang="ja">てきている</span></b><span lang="ja">。</span>
她的脸色一直在逐渐变得苍白。

<span lang="ja">時間はあっという間に過ぎ</span><b><span lang="ja">ていった</span></b><span lang="ja">。</span>
时间转瞬即逝。

<span lang="ja">この世界は、少しずつ変わっ</span><b><span lang="ja">ていっている</span></b><span lang="ja">。</span>
这个世界正在一点一滴地持续变化着。
</pre>

有些动词**要求**搭配方向性助动词才能使句子语法通顺。例如动词<span lang="ja">引っ越す</span>意为"搬家"，但与英语不同，它本身并没有"运动"或动态的含义。当你想用它来表达"我刚搬到这里"并突出状态转换时，必须使用方向助动词。其他动词如<span lang="ja">送る</span>和<span lang="ja">歩く</span>也有类似的情况。

<pre>
<span lang="ja">親の都合で、この街に引っ越し</span><b><span lang="ja">てきた</span></b>
因为父母的情况，他搬到了这个城市。

<span lang="ja">メールを送っ</span><b><span lang="ja">てきた</span></b><span lang="ja">人物は、結局何者だったのか。</span>
发了邮件的那个人，说到底到底是谁呢？

<span lang="ja">一人の少女がこちらに歩い</span><b><span lang="ja">てきた</span></b><span lang="ja">。</span>
一个独自的少女朝我走了过来。
</pre>
