# 用<span lang="ja">〜たり〜たり</span>和<span lang="ja">ては</span>进行列举和反复

我们在[第36课](../Part3/Lesson36.md)中已经看到，可以使用列举助词<span lang="ja">とか</span>来制作动词的**非穷尽**列表。

<span lang="ja">たり</span>是另一个可以以类似方式使用的助词/活用形式。实际上，用这种方式比用<span lang="ja">とか</span>更常见、更口语化。

动词的<span lang="ja">たり</span>形就是过去式 + <span lang="ja">り</span>：

| 原形 | 过去式 | <span lang="ja">たり</span>形 |
|------------|--------|-----------|
| <span lang="ja">食べる</span>     | <span lang="ja">食べた</span> | <span lang="ja">食べたり</span>  |
| <span lang="ja">飲む</span>       | <span lang="ja">飲んだ</span> | <span lang="ja">飲んだり</span>  |
| <span lang="ja">切る</span>       | <span lang="ja">切った</span> | <span lang="ja">切ったり</span>  |
| <span lang="ja">死ぬ</span>       | <span lang="ja">死んだ</span> | <span lang="ja">死んだり</span>  |

在<span lang="ja">〜たり</span>动作列表的末尾，通常会接上动词<span lang="ja">する</span>，这将整个动词块变成一个可活用的短语，表现得像一个大动词。

<pre>
<span lang="ja">手をつない</span><b><span lang="ja">だり</span></b><span lang="ja">、抱きしめあっ</span><b><span lang="ja">たり</span></b><span lang="ja">、キスをし</span><b><span lang="ja">たり</span></b><span lang="ja">したいから。</span>
因为我想做<b>比如</b>牵手、互相拥抱、接吻<b>之类的事</b>。
</pre>

与使用<span lang="ja">て</span>形连接动词不同，<span lang="ja">たり</span>不假定任何严格的顺序或动作的先后关系。它只是陈述这些动作中的任何一个（甚至其他未提及的动作）以任意顺序发生。A<span lang="ja">て</span>B大致表示"做A，然后做B"，而A<span lang="ja">たり</span>B<span lang="ja">たりする</span>表示"做像A和B之类的事"。

除了最基本的解释之外，它还有一些特定的惯用用法。它可以用来：

- 软化陈述；
- 为你所说的话添加一种意在让听者感到惊讶的细微含义；
- 陈述一种常见现象；
- 陈述一种可能的假设。

<span lang="ja">たり</span>的这些惯用用法有时会重叠，因此并不容易精确地指出哪种用法对应哪种含义。

就像其他非穷尽列举助词一样，它也可以只用列表中的一个元素。

<pre>
<span lang="ja">笑っ</span><b><span lang="ja">たりして</span></b><span lang="ja">ごめんなさい。</span>
抱歉我笑了。

<span lang="ja">実はそう</span><b><span lang="ja">だったりする</span></b>
事实上，情况确实如此。

<span lang="ja">昔はよく一緒に遊ん</span><b><span lang="ja">だりしました</span>.</b>
我们过去经常一起玩耍。

<span lang="ja">もしかしてお金が必要</span><b><span lang="ja">だったりする</span></b><span lang="ja">のだろうか？</span>
难道你也许需要一些钱吗？
</pre>

<span lang="ja">〜たり〜たり</span>也可以用来突出两个交替反复发生的动作。在这种用法中，两个动作往往是天然的对立面，并且在这个<span lang="ja">たり</span>组的末尾不使用<span lang="ja">する</span>动词。

<pre>
<span lang="ja">大きくなっ</span><b><span lang="ja">たり</span></b><span lang="ja">小さくなっ</span><b><span lang="ja">たり</span></b><span lang="ja">。</span>
它变大了，然后又变小了，然后又变大了，然后又变小了……
</pre>

<span lang="ja">ては</span>这种活用形式也可以以类似的方式使用。

<pre>
<span lang="ja">飲ん</span><b><span lang="ja">では</span></b><span lang="ja">笑い、笑っ</span><b><span lang="ja">ては</span></b><span lang="ja">飲む。</span>
他喝然后笑，又笑然后喝。
</pre>
