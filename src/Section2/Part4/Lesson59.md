# <span lang="ja">だって</span>的口语用法

除了（<span lang="ja">だ</span>）<span lang="ja">って</span>的引述用法之外，还有另外两种用法。其中一种是引述用法的延伸，但另一种在历史上是无关联的。

感叹性<span lang="ja">だって</span>出现在句子前面，用作一个转折感叹词，类似于口语化的<span lang="ja">しかし</span>，或者作为对前一句的解释/逻辑延续。

<pre>
<span lang="ja">「早く宿題しなさい！」</span>
<span lang="ja">「</span><b><span lang="ja">だって</span></b><span lang="ja">、つかれてるんだもん」</span>
"快点做作业！"
"<b>可是</b>我好累啊……"

<b><span lang="ja">だって</span></b><span lang="ja">おかしいじゃないですか。</span>
<b>我的意思是</b>，你不觉得很奇怪吗？
</pre>

另一种<span lang="ja">だって</span>是当用于名词后表示"甚至"含义时，作为<span lang="ja">も</span>/<span lang="ja">でも</span>的更口语化版本。

<pre>
<span lang="ja">そんなことは俺</span><b><span lang="ja">だって</span></b><span lang="ja">分かっている。</span>
<b>连</b>我也明白那种事
</pre>

这种<span lang="ja">だって</span>也可以以<span lang="ja">たって</span>的形式替换动词和形容词的<span lang="ja">ても</span>活用形。听起来非常口语化。

<pre>
<span lang="ja">どんなものを食べ</span><b><span lang="ja">たって</span></b><span lang="ja">問題ないな</span>
不管我吃什么东西都没问题啊

<span lang="ja">別に強くなく</span><b><span lang="ja">たって</span></b><span lang="ja">いい。</span>
<b>就算</b>你不强也没关系。
</pre>
