# 疑问词修饰：<span lang="ja">か</span>、<span lang="ja">も</span> 和 <span lang="ja">でも</span>

除了标记疑问之外，<span lang="ja">か</span>还可以用来将疑问词变成泛指名词。

|            |               |
|------------|---------------|
| <span lang="ja">なに</span>       | 什么          |
| <span lang="ja">なに</span>**<span lang="ja">か</span>** | **某**物 |

<br />
 
|            |             |
|------------|-------------|
| <span lang="ja">だれ</span>       | 谁         |
| <span lang="ja">だれ</span>**<span lang="ja">か</span>** | **某**人 |

<br />

|            |               |
|------------|---------------|
| <span lang="ja">どこ</span>       | 哪里         |
| <span lang="ja">どこ</span>**<span lang="ja">か</span>** | **某**处 |

<br />

|            |                           |
|------------|---------------------------|
| <span lang="ja">いつ</span>       | 何时                      |
| <span lang="ja">いつ</span>**<span lang="ja">か</span>** | **某**个时候 |

<span lang="ja">なにか</span>经常含糊地说成<span lang="ja">なんか</span>：

<pre>
<b><span lang="ja">なんか</span></b><span lang="ja">飲む？</span>
要喝点<b>什么</b>吗？
</pre>

请确保不要混淆以下两种用法：

<pre>
<span lang="ja">何</span><b><span lang="ja">が</span></b><span lang="ja">あった？</span>  
发生了什么？（字面：有什么？）

<span lang="ja">何</span><b><span lang="ja">か</span></b><span lang="ja">あった？</span>  
发生了什么事情吗？（字面：有什么东西吗？）
</pre>

注意，疑问词 + <span lang="ja">か</span>后面常常省略<span lang="ja">が</span>助词（<span lang="ja">何かが</span> -> <span lang="ja">何か</span>），可能是因为这样更容易发音。

同样地，我们也可以使用<span lang="ja">も</span>将疑问词变成"包含/排除"性名词。可以类比英语中的"anything"或"nothing"这样的词。

|        |                          |
|--------|--------------------------|
| <span lang="ja">なにも</span> | 任何东西/什么都没有         |
| <span lang="ja">だれも</span> | 每个人/任何人/没有人   |
| <span lang="ja">どれも</span> | 任何一个/没有一个 |
| <span lang="ja">いつも</span> | 任何时候/从不/总是     |

<pre>
<span lang="ja">ここは何</span><b><span lang="ja">も</span></b><span lang="ja">ない</span>  
这里<b>什么也</b>没有

<span lang="ja">誰</span><b><span lang="ja">も</span></b><span lang="ja">が知ってること</span>  
<b>每个人</b>/<b>任何人</b>都知道的事情。

<span lang="ja">いつ</span><b><span lang="ja">も</span></b><span lang="ja">そんなことを考えてるの？</span>  
你<b>一直</b>在想着那种事吗？
</pre>

注意：当<span lang="ja">誰も</span>在非否定句中作主语时，<span lang="ja">が</span>不能省略，所以会变成<span lang="ja">誰もが</span>。

在"每一个特定的事物"和"任意一个特定的事物"之间存在逻辑上的区别。通过将<span lang="ja">でも</span>附加到其中一些词上，可以赋予它们"无论哪一个"的"任意"含义。

<pre>
<span lang="ja">あんたは誰</span><b><span lang="ja">でも</span></b><span lang="ja">助けるの？</span>  
你真的会救<b>任何人</b>（无论谁）吗？

<span lang="ja">場所はどこ</span><b><span lang="ja">でも</span></b><span lang="ja">いい。</span>  
<b>任何地方</b>（无论哪里）都可以。
</pre>
