# 推测与传闻：<span lang="ja">みたい</span>、<span lang="ja">らしい</span>、<span lang="ja">ぽい</span> 和 <span lang="ja">そう</span>

日语中有很多方式来表示"X像Y"或"X看起来像Y"。

其中之一是<span lang="ja">な</span>形容词后缀<span lang="ja">みたい</span>。它看起来像<span lang="ja">見る</span>的<span lang="ja">たい</span>形 \-\> <span lang="ja">見たい</span>，然而在现代日语中，这被认为是完全不同的东西，不应混淆。

<span lang="ja">みたい</span>（它**永远**用假名书写，**绝对不能**写为<span lang="ja">見たい</span>）是一个<span lang="ja">な</span>-形容词，因此按<span lang="ja">な</span>-形容词变形。它的意思是"似乎"。

<pre>
<span lang="ja">あの人、日本人</span><b><span lang="ja">みたい</span></b>
那个人，<b>好像</b>是日本人 / 他看起来像日本人

<span lang="ja">ヒーロー</span><b><span lang="ja">みたいな</span></b><span lang="ja">人</span>
<b>像</b>英雄一样的人
</pre>

虽然<span lang="ja">みたい</span>在几乎所有意图和目的上都是一个<span lang="ja">な</span>形容词，但当用作副词时，它有时会像<span lang="ja">い</span>形容词一样变形。这被视为随意/俚语，但你在日本媒体中会经常看到，所以值得注意。

<pre>
<span lang="ja">子供</span><b><span lang="ja">みたいに</span></b><span lang="ja">ブランコに乗っている</span>
<span lang="ja">子供</span><b><span lang="ja">みたく</span></b><span lang="ja">ブランコに乗っている</span>
他像小孩一样坐在秋千上。
</pre>

这两个例子都是有效的，但前者更规范。

<span lang="ja">らしい</span>是一个辅助形容词，附加在名词或动词/形容词上，表示某事根据情况在事实上"似乎是……"。<span lang="ja">らしい</span>适用于可以翻译为"似乎是X"而非"有点像X"的事物。

<pre>
<span lang="ja">日本人</span><b><span lang="ja">らしい</span></b><span lang="ja">名前。</span>
一个<b>像是</b>日本人的名字。
</pre>

你也可能看到<span lang="ja">らしい</span>用来表示"我听说"，用于传达传闻或从第三方获得的信息。

<pre>
<span lang="ja">出発は明日の昼頃になる</span><b><span lang="ja">らしい</span></b><span lang="ja">。</span>
<b>听说</b>出发时间会是明天中午左右。
</pre>

<span lang="ja">ぽい</span>/<span lang="ja">っぽい</span>是一个后缀<span lang="ja">い</span>形容词，表示某事基于外观或刻板属性主观上"似乎如此"。它可以很好地翻译为"有点X的样子"或"像X的"。听起来很随意。

<pre>
<span lang="ja">今、僕の名前、女</span><b><span lang="ja">っぽい</span></b><span lang="ja">って思ったでしょ？</span>
你刚才觉得我的名字<b>有点</b>女性<b>-化</b>吧？（字面："女性-似的"）
</pre>

辅助后缀<span lang="ja">そう</span>有两种不同的结构。

第一种附加在动词或形容词的连用形（masu词干）上，意思是"看起来……"。它用于根据主观印象或视角来描述事物，通常是从外观上来看。<span lang="ja">そう</span>将其附加的任何词变成一个<span lang="ja">な</span>形容词。

<pre>
<span lang="ja">おいし</span><b><span lang="ja">そう</span></b><span lang="ja">なスープです。</span>
一份<b>看起来很好吃</b>的汤。
</pre>

<div class="warning">
不要将<span lang="ja">かわいそう</span>（可怜的）与<span lang="ja">かわいい</span>（可爱的）的<span lang="ja">そう</span>版本混淆。<span lang="ja">かわいい</span>没有<span lang="ja">そう</span>版本。如果你想表达某物看起来很可爱，可以用<span lang="ja">かわいくみえる</span>来代替。
</div>

另一种<span lang="ja">そう</span>附加在词的完整形式之后，所以对于<span lang="ja">な</span>形容词和名词会变成<span lang="ja">だそう</span>。这种用法表示"我听说"，类似于我们之前看到的<span lang="ja">らしい</span>的传闻用法。它感觉更正式一些。

<pre>
2000<span lang="ja">人以上が死んだ</span><b><span lang="ja">そう</span></b><span lang="ja">です。</span>
<b>听说</b>有2000多人遇难了。
</pre>
