# 转折：<span lang="ja">が</span>、<span lang="ja">けど</span>、<span lang="ja">しかし</span> 和 <span lang="ja">ても</span>/<span lang="ja">でも</span>

与**主语标记**<span lang="ja">が</span>无关，**连接词**<span lang="ja">が</span>表示，即使前面的陈述为真，后面的陈述也为真。使用<span lang="ja">が</span>时，两个陈述都是独立的从句。通常（但并非总是），第二个陈述与第一个陈述形成对比。这通常被翻译为"但是"或"虽然"，但在某些情况下，它仅仅是一个"而且"，完全没有对比。

<pre>
<span lang="ja">少し早い</span><b><span lang="ja">ですが</span></b><span lang="ja">、おめでとうございます。</span>
稍微早了一点<b>不过</b>，恭喜你。
</pre>

<span lang="ja">が</span>附加在完整的句子之后，这意味着在名词和<span lang="ja">な</span>形容词之后**需要**<span lang="ja">だ</span>（或<span lang="ja">です</span>）。<span lang="ja">ですが</span>在口语和日常礼貌用语中很常见，但<span lang="ja">だが</span>听起来太生硬，几乎只限于书面语或非常正式的语言。

与<span lang="ja">が</span>一样，还有连接词<span lang="ja">けど</span>。它们的用法和含义基本上可以视为相同。就像<span lang="ja">が</span>一样，<span lang="ja">ですけど</span>和<span lang="ja">だけど</span>都存在，在现实对话语气中也非常常见。<span lang="ja">けど</span>比<span lang="ja">が</span>显得不那么正式。

<pre>
<span lang="ja">しばらく考えてみた</span><b><span lang="ja">けど</span></b><span lang="ja">、何も思い浮かばなかった。</span>
我试着想了一会儿<b>可是</b>，什么也想不出来。
</pre>

<span lang="ja">けど</span>和<span lang="ja">が</span>都涵盖了可能感觉非常微妙的正式/礼貌空间，所以这里有一些经验法则。这些不是实际的规则，只是一般建议：

- 如果整个句子是简体形式，使用（<span lang="ja">だ</span>）<span lang="ja">けど</span>
- 如果是会话语言，不要使用（<span lang="ja">だ</span>）<span lang="ja">が</span>
- 如果整个句子是敬体形式，使用<span lang="ja">ですが</span>
- 以敬体形式结束陈述但在中间使用<span lang="ja">けど</span>是常见的，但如果礼貌程度不适当，可能会显得不自然

<span lang="ja">が</span>/<span lang="ja">けど</span>的一个非常常见的用法是以间接的方式引入某事物作为话题。在这种用法中，它没有转折（"但是"）的语感。它只是在谈论某事物之前先引入它。这种用法常与说明性的<span lang="ja">の</span>助词一起出现，如<span lang="ja">なんだけど</span>或<span lang="ja">なんですが</span>：

<pre>
<span lang="ja">「もしもし、ネミアちゃん？　俺</span><b><span lang="ja">だけど</span></b><span lang="ja">、どうしたの？」</span>
（在电话中）"喂，Nemia？<b>是我</b>，怎么了？"
</pre>

这种用法在缓和气氛以便向听者提出请求或问题时非常常见：

<pre>
<span lang="ja">一つお願いがある</span><b><span lang="ja">んですが</span></b><span lang="ja">、よろしいでしょうか？</span>
我有一个请求<b>……</b>可以吗？
</pre>

这种短语如此常见，以至于在很多情况下，说话者甚至不需要在<span lang="ja">んですが</span>/<span lang="ja">んだけど</span>之后提供解释，以至于句子的后半部分可以完全省略。

<pre>
<span lang="ja">すいません、本を探している</span><b><span lang="ja">んですが</span></b>
（在店里）不好意思，我在找一本书<b>……</b>（你能帮我吗？）
</pre>

<span lang="ja">けど</span>也可以以<span lang="ja">けれど</span>和<span lang="ja">けれども</span>的形式出现。它们*基本上*是一样的，但越长越正式。

<span lang="ja">しかし</span>是一个感叹词。它听起来很正式。说英语的人往往过度使用这个词，因为它类似于我们在自己语言中在句首使用"but"（但是）的方式，但在日语中它并不那么常见。

<pre>
<b><span lang="ja">しかし</span></b><span lang="ja">、知らなかった。</span>
<b>然而</b>，我不知道。
</pre>

<span lang="ja">ても</span>是<span lang="ja">て</span>形和<span lang="ja">も</span>的组合。它作为一个连接词，表示后面的陈述为真，*尽管*前面的陈述如此。这类似于转折，但不总是转折。

<pre>
<span lang="ja">そんな事言わなく</span><b><span lang="ja">ても</span></b><span lang="ja">わかっているんだろ？</span>
<b>即使</b>我不说那种事，你也明白吧？
</pre>

还有<span lang="ja">でも</span>，可以在新句子的开头添加"但是"的转折含义。

<pre>
<b><span lang="ja">でも</span></b><span lang="ja">、あなたは分かっているでしょ？</span>
<b>但是</b>，你明白的吧？
</pre>
