# 疑问词：<span lang="ja">なに</span>、<span lang="ja">だれ</span>、<span lang="ja">どれ</span>、<span lang="ja">いつ</span> 等

这些被称为疑问词。它们用于提问特殊疑问句（wh-问句）。有时它们也会作为更大的语法结构或固定表达的一部分出现。

| 汉字   | 读法       | 含义  | 例句（日文）                  | 例句（中文）               |
|------|-------------|---------|--------------------------|---------------------------------|
| <span lang="ja">何</span>   | <span lang="ja">なに</span>        | 什么    | <span lang="ja">何する？</span>                 | 你要做什么？             |
| <span lang="ja">誰</span>   | <span lang="ja">だれ</span>        | 谁     | <span lang="ja">あなた、誰？</span>             | 你是谁？                |
|      | <span lang="ja">どれ</span>        | 哪个   | <span lang="ja">どれがいいと思いますか？</span> | 你觉得哪个好？ |
|      | <span lang="ja">いつ</span>        | 何时    | <span lang="ja">次の電車はいつですか？</span>   | 下一班电车是什么时候？   |

还有很多其他的疑问词，它们的用法都差不多，所以我们不会全部列出。如果你还记得<span lang="ja">こそあど</span>词的课程，<span lang="ja">こそあど</span>系列中的<span lang="ja">ど</span>部分指的就是这些词。

在提问时，需要记住的一点是，**绝对不要**用助词<span lang="ja">は</span>来标记这些词。唯一的例外是一些固定表达，如<span lang="ja">何はともあれ</span>，但这些基本上屈指可数。

<pre>
<span lang="ja">昨日は誰</span><b><span lang="ja">が</span></b><span lang="ja">来たの？</span>
昨天谁来了？
</pre>
