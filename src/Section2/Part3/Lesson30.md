# 引用：<span lang="ja">と</span>、<span lang="ja">って</span> 和 <span lang="ja">という</span>

有两种基本类型的关系从句。第一种是修饰名词的类型，我们已经在[第14课](../../Section1/Part1/Lesson14.md)中学过了。第二种是嵌入从句，通常被称为"引用"，就像句子"He said that <ins>the war would end soon</ins>"（他说<ins>战争很快就会结束</ins>）中带下划线的部分。

<span lang="ja">って</span> 和 <span lang="ja">と</span> 被称为引用标记，通常承担标记这种嵌入从句的角色。

<pre>
<span lang="ja">いい</span><b><span lang="ja">と</span></b><span lang="ja">思う</span> 
我觉得挺好的。

<span lang="ja">見える</span><b><span lang="ja">と</span></b><span lang="ja">聞いた</span>
我听说可以看见。

<span lang="ja">怖くない</span><b><span lang="ja">と</span></b><span lang="ja">言った</span>
我说了我不怕。
</pre>

在英语中，我们通过标点符号来区分直接引语和间接引语。例如：

<pre>
"I will eat the cake," he said. （直接引语）
He said he would eat the cake. （间接引语）
</pre>

同样的概念不适用于日语，在日语中，直接引语和间接引语之间的区别更加模糊。虽然有引号（如 <span lang="ja">「　」</span>）存在，可以帮助识别直接引语，但它们不是必需的，你会看到直接引语和间接引语都"自由地"出现在文本中，仅由一个简单的<span lang="ja">って</span>或<span lang="ja">と</span>标记。

引语中的嵌入从句需要是一个"完整"的陈述，所以如果引语以<span lang="ja">な</span>形容词或名词结尾，就需要以<span lang="ja">だ</span>结尾。有一些例外和用法可能会省略这个规则，但在大多数情况下，你需要确保<span lang="ja">だ</span>存在，否则句子听起来会不对。

<pre>
<span lang="ja">優しい人</span><b><span lang="ja">だと</span></b><span lang="ja">思う</span>  
我觉得他是个善良的人。
</pre>

在引用方面，<span lang="ja">って</span>本质上是<span lang="ja">と</span>的口语版本。

<pre>
<span lang="ja">ダメ</span><b><span lang="ja">だって</span></b><span lang="ja">言ったでしょ！</span>
我说了不行，不是吗？

<span lang="ja">何</span><b><span lang="ja">だって</span></b><span lang="ja">？</span>
你说什么？
</pre>

从第二个例子可以看出，动词（"说"、"想"等）如果从上下文和语气中很明显的话，通常可以省略。

在某些用法中，<span lang="ja">って</span>也可以承担话题的角色，类似于<span lang="ja">は</span>。在这些情况下，它并不是真正的引用。它只是一个替代的话题标记。

<pre>
<span lang="ja">田中さん</span><b><span lang="ja">って</span></b><span lang="ja">いい人だね</span>   
田中先生是个好人啊。
</pre>

构造<span lang="ja">という</span>（经常含糊地说成<span lang="ja">っていう</span>，甚至<span lang="ja">っつう</span>）使用得如此频繁，以至于它在字面意思"我/他们说"的基础上获得了额外的含义。

它可以用一种使句子更流畅、不那么生硬的方式来"粘合"句子的各个部分。很难为所有这些情况提供一个全面的解释，但你会在足够多的场合中看到它，所以通过接触来获得一种"感觉"会更好。有时<span lang="ja">という</span>根本不添加任何含义，所以如果没法翻译成中文，不要勉强。

它可以用来提供定义或描述某个属性。这种用法类似于<span lang="ja">である</span>，但语感略有不同。

<pre>
<span lang="ja">研究者という人間</span>  
那些被称为研究者的人（作为研究者而被认定的人）

<span lang="ja">研究者である人間</span>  
那些身为研究者的人
</pre>

它也可以在一些固定表达中通过重复来提供强调：

<pre>
<span lang="ja">今日という今日</span>
就在今天这一天

<span lang="ja">木という木</span>
每一棵树
</pre>
