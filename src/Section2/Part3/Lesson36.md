# 列举：<span lang="ja">や</span>、<span lang="ja">とか</span>、<span lang="ja">など</span>、<span lang="ja">と</span>、<span lang="ja">か</span> 和 <span lang="ja">に</span>

日语中的项目列表是通过在每个项目上附加一个特定的"列举"助词来构成的，这个助词定义了列表的性质。列表可以是穷举的，也可以是非穷举的。穷举列表是明确列出其所包含的每一个元素的列表。非穷举列表是提供几个例子或描述其元素性质，同时留有模糊和间接空间的列表。

"我和Paul和Frank一起去了商场"是一个穷举列表，而"我和Paul、Frank以及其他同伴一起去了商场"则是一个非穷举列表。

根据列表的性质，最后一个元素的列举助词通常会省略，并替换为适用于整个列表的格助词。

<span lang="ja">や</span>构成一个**非穷举**列表，提供几个元素的例子，这些元素与它们的一个共同概念相关。

<pre>
<span lang="ja">スプーン</span><b><span lang="ja">や</span></b><span lang="ja">フォークで食べること</span>
用勺子<b>和</b>叉子（之类的东西）来吃
</pre>

<span lang="ja">と</span>之前作为附带说明提过，但这里我们将明确介绍它并与<span lang="ja">や</span>进行比较。<span lang="ja">と</span>构成一个"通用的"**穷举**列表。它列举列表中的每一个元素。

<pre>
<span lang="ja">リンゴ</span><b><span lang="ja">と</span></b><span lang="ja">魚</span><b><span lang="ja">と</span></b><span lang="ja">タイヤキ</span>(<b><span lang="ja">と</span></b>)<span lang="ja">を食べた</span>
我吃了鲷鱼烧、鱼<b>和</b>苹果。
</pre>

如前所述，最后一个<span lang="ja">と</span>是可选的，通常会被省略。

<span lang="ja">か</span>可以用来提供选项。它更接近"或"而不是"和"，尽管这种区分有时很模糊。

<pre>
<span lang="ja">スプーン</span><b><span lang="ja">か</span></b><span lang="ja">フォークで食べる。</span>
用勺子<b>或</b>叉子吃。
</pre>

<span lang="ja">とか</span>构成一个"模糊"的列表。它是模糊的，因此是**非穷举**的，但每个项目是独立的。这些项目不一定被认为共享共同属性。

<pre>
<span lang="ja">スプーン</span><b><span lang="ja">とか</span></b><span lang="ja">フォークで食べること</span>
用勺子<b>或者</b>叉子（或类似的东西）来吃。
</pre>

它也可以仅用一个元素来使事物变得模糊：

<pre>
<span lang="ja">例えばデスクトップパソコン</span><b><span lang="ja">とか</span></b><span lang="ja">。</span>
例如，<b>像</b>台式电脑<b>之类的</b>……
</pre>

毫不意外，<span lang="ja">とか</span>基本上就是<span lang="ja">と</span> + <span lang="ja">か</span>。它也可以用来列举动词，通常是为了提供替代选择。这是本课中其他列举助词所不具备的独特特点。

<pre>
<span lang="ja">戦える</span><b><span lang="ja">とか</span></b><span lang="ja">戦えない</span><b><span lang="ja">とか</span></b><span lang="ja">、そういう問題じゃないんだ。</span>
问题不在于你能打<b>还是</b>不能打。

<span lang="ja">もしかして、付き合ってる</span><b><span lang="ja">とか</span></b><span lang="ja">？</span>
难道说，你在交往<b>之类的</b>？
</pre>

如果列表太长（超过2-3个元素），通常会用逗号来分隔中间的元素，而不使用助词。

<span lang="ja">など</span>从技术上讲不是一个列举助词，而更像是一个填充助词，听起来像英语中使用"etc"（等等）或"and such"（之类的）。它暗示一个元素列表为更多类似的东西敞开。其含义类似于使用单个<span lang="ja">や</span>或<span lang="ja">とか</span>来表示暗示了更多例子但没有直接说明。

<pre>
<span lang="ja">完璧な人間</span><b><span lang="ja">など</span></b><span lang="ja">、存在しない。</span>
<b>像</b>完美的人类<b>之类的东西</b>是不存在的。
</pre>

这些列举助词也常常一起使用，尤其是与<span lang="ja">など</span>。模式X<span lang="ja">や、</span>Y<span lang="ja">、</span>Z<span lang="ja">など</span>是非常常见的，如下面的句子所示：

<pre>
<span lang="ja">剣</span><b><span lang="ja">や</span></b><span lang="ja">斧</span><b><span lang="ja">、</span></b><span lang="ja">槍</span><b><span lang="ja">、</span></b><span lang="ja">弓</span><b><span lang="ja">など、</span></b><span lang="ja">様々な武器が置かれている。</span>  
剑、斧、枪、弓<b>等等</b>，各种武器陈列着。
</pre>

还有一个列举助词有时会被忽略：<span lang="ja">に</span>。这个助词用于列举成套出现的事物，或者说话者以某种思路串联或主动回忆的方式逐条列举的事物。

<pre>
<span lang="ja">サンドイッチ</span><b><span lang="ja">に</span></b><span lang="ja">ミルク</span><b><span lang="ja">に</span></b><span lang="ja">コーヒー。</span>
三明治……牛奶……然后咖啡。
</pre>
