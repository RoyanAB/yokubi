# 连接词：<span lang="ja">のに</span>、<span lang="ja">ように</span>、<span lang="ja">ために</span>、<span lang="ja">せいで</span> 和 <span lang="ja">おかげで</span>

<span lang="ja">のに</span>与<span lang="ja">ので</span>相关，但它的作用方向相反。虽然<span lang="ja">ので</span>表示"因为X，所以Y"，但<span lang="ja">のに</span>表示"尽管X，却Y"。像<span lang="ja">ので</span>一样，这比其他转折方式更"客观"。<span lang="ja">のに</span>也喜欢附加在句末，作为失望或不满的强调标记。

<pre>
<span lang="ja">もう会えないと思ってた</span><b><span lang="ja">のに、</span></b><span lang="ja">また会うことができた。</span>
<b>明明</b>我以为再也见不到你了，却又见到了。
</pre>

和<span lang="ja">ので</span>一样，<span lang="ja">のに</span>在名词和<span lang="ja">な</span>形容词之后需要<span lang="ja">な</span>

<pre>
<span lang="ja">少し考えれば、当たり前のこと</span><b><span lang="ja">なのに</span></b><span lang="ja">。</span>
稍微想一想就会发现，这是理所当然的事情嘛……（带有失望的语气）
</pre>

<span lang="ja">ように</span>基本上意思是"以便"，如"walk so as to keep your back hidden"（走路以便隐藏你的背部）。这在"为了……目的"和"以……方式"之间微妙地徘徊。

<pre>
<span lang="ja">足音を立てない</span><b><span lang="ja">ように</span></b><span lang="ja">歩く</span>
走路<b>以</b>不发出脚步声。
</pre>

如果有助于理解，你可以把<span lang="ja">ように</span>看作是一个名词（<span lang="ja">よう</span> = "方式"）通过<span lang="ja">に</span>变成了副词。修饰<span lang="ja">よう</span>的从句成为主动词的"副词性"修饰。

<span lang="ja">ために</span>字面上使用名词<span lang="ja">ため</span>"目的"加<span lang="ja">に</span>来构成连接词。

<pre>
<span lang="ja">それを確かめる</span><b><span lang="ja">ために、</span></b><span lang="ja">彼は走っていた。</span>  
他正在奔跑<b>，为了</b>确认那件事。
</pre>

在很多情况下，<span lang="ja">ために</span>和<span lang="ja">ように</span>在表示"为了……目的"时可以互换，但不总是如此。

<span lang="ja">ために</span>也可以用来表示"为了……的缘故"而不是"为了……的目的"。在英语中语感不同，但在日语中并不总是那么明显。<span lang="ja">に</span>也可以省略。

<pre>
<span lang="ja">あなたの</span><b><span lang="ja">ため</span></b><span lang="ja">なら、なんでもするよ。</span>
<b>为了</b>你，我什么都愿意做。
</pre>

<span lang="ja">せい</span>是一个名词，意思是"过错"，用于指示原因/责备/理由。它通常暗示结果是不好的。

<pre>
<span lang="ja">お前の</span><b><span lang="ja">せいで</span></b><span lang="ja">俺はこんなことになったんだよ！</span>
<b>都是因为你</b>，我才变成这样的！
</pre>

<span lang="ja">せいで</span>的反义词是<span lang="ja">おかげで</span>，它暗示一种感激之情。

<pre>
<span lang="ja">俺は、お前の</span><b><span lang="ja">おかげで</span></b><span lang="ja">救われたんだ。</span>
<b>多亏了</b>你，我才能得救。
</pre>
