# 委婉请求：<span lang="ja">なさい</span> 和禁止：<span lang="ja">な</span>

<span lang="ja">なさい</span>表示请求某人做某事。它给人的感觉像是说话者期待听者去做这个动作，但仍然是一个温和的请求。它有一种间接/柔和的意味，虽然仍然是相当性别中性的，但常常被视为女性化的用法。说话者的地位通常高于听者，常见于老师和母亲训斥小孩子时。你也可以在常见的固定表达中看到它，如<span lang="ja">おやすみなさい</span>，在这种情况下没有额外的语感。

<pre>
<span lang="ja">聞き</span><b><span lang="ja">なさい</span></b>
请听好。

<span lang="ja">食べ</span><b><span lang="ja">なさい</span></b> 
请吃吧。
</pre>

最后，我们来看用来表示禁止的<span lang="ja">な</span>。它附加在不带<span lang="ja">ます</span>、非过去时的陈述末尾。这个句末助词可能会与[第29课](./Lesson29.md)中提到的<span lang="ja">な</span>句末助词混淆。上下文和语气会告诉你哪个是哪个。它们非常不同。

<pre>
<span lang="ja">来る</span><b><span lang="ja">な</span></b><span lang="ja">！</span>
别过来！（不要来！）
</pre>

注意：<span lang="ja">なさい</span>中的<span lang="ja">さい</span>部分也常常被省略。你可以很容易地将它与禁止的<span lang="ja">な</span>区分开来，因为它附加在动词的连用形（masu词干）上：

<pre>
<span lang="ja">聞き</span><b><span lang="ja">な</span></b>
请听

<span lang="ja">聞く</span><b><span lang="ja">な</span></b>！
别听！
</pre>
