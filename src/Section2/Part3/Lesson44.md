# 禁止与必须：<span lang="ja">だめ</span>、<span lang="ja">いけない</span>、<span lang="ja">ならない</span> 以及如何说"必须"

<span lang="ja">だめ</span>是一个表示"不行"、"没用"或"白费力气"的词。它可以用来表达某事不好，通过使用<span lang="ja">ては</span>或条件形式如<span lang="ja">たら</span>等来连接前面的陈述。

<pre>
<span lang="ja">あまり迷惑をかけ</span><b><span lang="ja">てはダメ</span></b><span lang="ja">ですよ。</span>  
给别人添太多麻烦<b>是不好的</b>。（字面："如果你给别人添太多麻烦，那是不好的"）

<span lang="ja">日本だっ</span><b><span lang="ja">たらダメ</span></b><span lang="ja">なんだろうけど、ここでは大丈夫みたい。</span>  
如果是在日本的话<b>就不行了</b>，但在这里似乎没问题。
</pre>

<span lang="ja">ならない</span>和<span lang="ja">いけない</span>类似于<span lang="ja">だめ</span>，但感觉更强，暗示某件事是被禁止的，无论是因为某人的决定还是社会期望。

<pre>
<span lang="ja">他人のものを、盗ん</span><b><span lang="ja">ではならない</span></b><span lang="ja">。</span>
偷别人的东西<b>是不允许的</b>。（字面："是不能做的事"）
</pre>

这三组"不好的"表达（<span lang="ja">だめ</span>、<span lang="ja">ならない</span>、<span lang="ja">いけない</span>）通常与否定条件从句（<span lang="ja">〜なければならない</span>、<span lang="ja">〜なくてはいけない</span>、<span lang="ja">〜ないとダメ</span>，以及任何其他类似的组合）一起使用，来表达"必须做"的意思。既然A<span lang="ja">てはならない</span>表示"不得做A"，那么逻辑上（非A）<span lang="ja">てはならない</span>就是"不得*不做*A"，因此就是"必须做A"。

这就是日语中说"必须做X"或"不得不做X"的方式。听起来很拗口，但你会很快习惯的。

<pre>
<span lang="ja">やら</span><b><span lang="ja">なければならない</span></b><span lang="ja">ことはたくさんある。</span>
有很多事<b>我必须做</b>。（字面："如果我不做就不可接受的事情"）
</pre>

<div class="warning">
从文体上来说，<span lang="ja">〜ないと</span> + <span lang="ja">ならない</span>的组合被视为糟糕的形式，有些人会认为这是不正确的。即使在书中，你也会相对频繁地遇到它，但建议避免使用它，转而选择<span lang="ja">〜ないといけない</span>。
</div>

<span lang="ja">なければ</span>通常缩写为<span lang="ja">なきゃ</span>，<span lang="ja">ては</span>通常缩写为<span lang="ja">ちゃ</span>（<span lang="ja">では</span>变成<span lang="ja">じゃ</span>）。

<pre>
<span lang="ja">死ん</span><b><span lang="ja">じゃ</span></b><span lang="ja">ダメ！</span>
你<b>不能</b>死！（字面："死了可不行"）
</pre>

当被上下文暗示时，在陈述的末尾，条件之后的部分通常会被省略，因为这些表达如此极其常见，大家都知道后面要表达什么。

<pre>
<span lang="ja">冷めないうちに食べ</span><b><span lang="ja">ないと</span></b><span lang="ja">。</span>
你<b>必须</b>趁着还没凉吃掉。
</pre>
