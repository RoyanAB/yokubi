# 同时进行：<span lang="ja">ながら</span>、<span lang="ja">あいだ</span>、<span lang="ja">うちに</span> 和 <span lang="ja">つつ</span>

本课重点介绍如何表达某事在进行的同时另一件事也在发生。

<span lang="ja">ながら</span>表示两个动作同时进行、同时为真。两个动作的主语**必须**相同。例如"我边吃薯片边看电视"是可以的，但"我边看电视，我弟弟边玩电子游戏"则不行。<span lang="ja">ながら</span>附加在动词的连用形（masu词干）上。

<span lang="ja">ながら</span>的形式是A<span lang="ja">ながら</span>B，其中B所描述的动作是"主要"动作，A所描述的动作是同时进行的"次要"动作。在英语中这种语感不太容易表现出来，所以我们来看两个例子：

<pre>
<span lang="ja">画面を見</span><b><span lang="ja">ながら</span></b><span lang="ja">勉強しよう</span>
<b>一边</b>看屏幕<b>一边</b>学习吧。

<span lang="ja">そこで勉強し</span><b><span lang="ja">ながら</span></b><span lang="ja">、爺ちゃんの帰りを待つ</span>
我在那里<b>一边</b>学习<b>一边</b>等爷爷回家。
</pre>

在第一个例子中，主要动作是学习，这是说话者想做的事情；而在第二个例子中，说话者专注于等待的动作，在等待的同时学习。

<span lang="ja">つつ</span>是另一个辅助词，用法与<span lang="ja">ながら</span>大致相同，只是听起来更正式，几乎只限于书面语句子。

<pre>
<span lang="ja">彼女はそう言い</span><b><span lang="ja">つつ</span></b><span lang="ja">、苦笑いを浮かべる。</span>
她<b>一边</b>那样说着<b>一边</b>，露出了苦笑。
</pre>

<span lang="ja">つつある</span>是<span lang="ja">つつ</span>的一种特殊用法，暗示一个动作正在持续发生，类似于更正式的<span lang="ja">ている</span>。

<pre>
<span lang="ja">時代は変わり</span><b><span lang="ja">つつあります</span></b><span lang="ja">。</span>  
时代正在不断变迁。
</pre>

然而，与<span lang="ja">ている</span>不同，它不能用于描述状态以表示"处于某种状态"：

<pre>
<span lang="ja">男はもう死に</span><b><span lang="ja">つつある</span></b><span lang="ja">。</span>
那个男人已经<b>正在死去</b>。

<span lang="ja">男はもう、死ん</span><b><span lang="ja">でいる</span></b><span lang="ja">。</span>
那个男人已经<b>死了</b>。
</pre>

<span lang="ja">間</span>/<span lang="ja">あいだ</span>（<span lang="ja">に</span>）和<span lang="ja">うちに</span>是另外两种表示某事在另一件事发生期间发生的方式，但是用法更通用，比<span lang="ja">ながら</span>和<span lang="ja">つつ</span>的范围更广。

<span lang="ja">間</span>/<span lang="ja">あいだ</span>表示两个事件覆盖完全相同的、**整体的**时间区间。这对它所接受的陈述类型施加了限制。<span lang="ja">間</span>本身像名词一样作用。

<pre>
<span lang="ja">二人がお風呂に入っている</span><b><span lang="ja">間</span></b><span lang="ja">、僕は色々考えた。</span>
那两个人洗澡<b>期间</b>，我想了很多事。
</pre>

A<span lang="ja">間</span>**<span lang="ja">に</span>**B表示B在A发生的时间段**之内**开始和结束。与不带<span lang="ja">に</span>的<span lang="ja">間</span>版本不同，它暗示B的持续时间比A的持续时间短，因为B需要完全包含在A的时间区间内。

<pre>
<span lang="ja">俺が意識を失っている</span><b><span lang="ja">間に</span></b><span lang="ja">何があったのだろうか？</span>
在我失去意识的<b>期间</b>，到底发生了什么呢？
</pre>

<span lang="ja">うちに</span>表示，在<span lang="ja">うちに</span>之前的陈述成立期间，发生了某个事件。

<pre>
<span lang="ja">そんな事を考えている</span><b><span lang="ja">うちに</span></b><span lang="ja">、完全に日が沈んでしまった。</span>
在想那些事的<b>期间</b>，太阳完全落下去了。
</pre>

它通常带有一种语感，即<span lang="ja">うちに</span>标记的事件在时间上是有限的，最终会结束。

<pre>
<span lang="ja">蜘蛛がいない</span><b><span lang="ja">うちに</span></b><span lang="ja">救出するか</span>
<b>趁着</b>蜘蛛不在，去救他们吗？
</pre>

这通常用于给出建议或提议，如"趁X发生之前，我们应该做Y"。

<pre>
<span lang="ja">冷めない</span><b><span lang="ja">うちに</span></b><span lang="ja">食べましょう。</span>
<b>趁</b>还没凉，快吃吧。（字面：趁还不冷的时候吃吧）
</pre>
