# 比较：<span lang="ja">より</span> 和 <span lang="ja">の方が</span>

<span lang="ja">より</span>用于直接比较两个事物，类似于"I'm smarter than you"（我比你聪明）中的"than"（比）。使用<span lang="ja">より</span>时，不需要使用"更"之类的词，这与英语不同——英语需要说"**more** X than Y"或"X**er** than Y"。有时<span lang="ja">より</span>可以后接<span lang="ja">も</span>来加强语气。

<pre>
<span lang="ja">彼は誰</span><b><span lang="ja">より</span></b><span lang="ja">も強いよ</span>
他<b>比</b>任何人都强
</pre>

在这个例子中，不带<span lang="ja">も</span>的<span lang="ja">誰より</span>会显得奇怪，不那么常见。

<span lang="ja">より</span>也用于关于相对位置（空间或时间上）的结构中，类似于<span lang="ja">から</span>。

<pre>
<span lang="ja">南</span><b><span lang="ja">より</span></b><span lang="ja">迫る</span>
<b>从</b>南边逼近

<span lang="ja">新世界</span><b><span lang="ja">より</span></b>
<b>来自</b>新世界
</pre>

这种<span lang="ja">より</span>的用法常见于书信中，其中<span lang="ja">より</span>标记寄信人，<span lang="ja">へ</span>标记收信人。

如果你想列出比较的双方，则用<span lang="ja">より</span>标记低的一方，用<span lang="ja">方</span>标记高的一方。<span lang="ja">方</span>在这种用法中读作<span lang="ja">ほう</span>（不要被<span lang="ja">かた</span>的读法迷惑了！），和所有名词一样，它通过<span lang="ja">の</span>与其它名词相连，并可以在后面接助词。在比较的情况下，无论是否是动作的主语，<span lang="ja">方</span>这一"边"通常由<span lang="ja">が</span>标记。

<pre>
<span lang="ja">メリット</span><b><span lang="ja">より</span></b><span lang="ja">もデメリット</span><b><span lang="ja">の方が</span></b><span lang="ja">大きい</span>
缺点<b>比</b>优点<b>更</b>大。
</pre>

<span lang="ja">より</span>那一方如果被暗示了则可以省略。

<pre>
<span lang="ja">私は一人で行動する</span><b><span lang="ja">方が</span></b><span lang="ja">性に合っている。</span>
我<b>更</b>适合一个人行动。
</pre>

从字面上看，这种<span lang="ja">方</span>的用法标记了比较的"一边"。X<span lang="ja">より</span>Y<span lang="ja">の方が</span>Z可以理解为"**从**X这一边来看，Y这一边**更**Z"，其中X是低的一方，Y是高的一方。

有一些惯用表达使用了<span lang="ja">方が</span>。

\<动词\><span lang="ja">方がいい</span>表示"最好做\<动词\>"。在这种结构中，如果陈述源于个人建议而非普遍真理，动词常使用过去时。

<pre>
<span lang="ja">やめ</span><b><span lang="ja">た方がいい</span></b><span lang="ja">と思います</span>
我觉得你<b>最好</b>停下来。

<span lang="ja">少し余裕があ</span><b><span lang="ja">る方がいい</span></b><span lang="ja">だろう。</span>
<b>最好还是</b>有点余裕吧。
</pre>

当<span lang="ja">いい</span>变为过去时如<span lang="ja">方がよかった</span>时，它暗示对过去应该做或不应该做的行为感到某种遗憾或责备。

<pre>
<span lang="ja">真面目に戦っ</span><b><span lang="ja">た方がよかった</span></b><span lang="ja">のだろうか。</span>
如果我认真战斗了，<b>会更好吗</b>？
</pre>
