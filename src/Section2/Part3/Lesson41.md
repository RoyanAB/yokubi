# 计数

现在让我们来谈谈计数。当你想表示数量时，通常不能在句子中间直接放"自由"的数字。你需要将它们与一个特殊的词配对，这个词描述你所定义的是什么类型的量。这些特殊的词被称为"量词"（counters），而且有**很多**。有些非常特定，会根据你计数的东西而变化，但有些更通用，规则也不像你可能被引导认为的那样严格。

这就像英语中你不能说"I eat 3 breads"（我吃3面包），而必须说"I eat 3 **loaves** of bread"（我吃3条面包）或"3 **slices** of bread"（3片面包）。日语对大多数可数事物都是这样运作的。

最常见的量词是<span lang="ja">つ</span>，但是<span lang="ja">つ</span>也很特殊，因为它不能超过9。如果你需要数超过9个"通用"的东西，你需要使用另一个量词（通常是<span lang="ja">個</span>，但你不能对所有东西都用<span lang="ja">個</span>）。

| 数字    | 读法                                              |
|--------|--------------------------------------------------|
| <span lang="ja">一つ　</span> | <span lang="ja">ひとつ</span>                                           |
| <span lang="ja">二つ　</span> | <span lang="ja">ふたつ</span>                                           |
| <span lang="ja">三つ　</span> | <span lang="ja">みっつ</span>                                           |
| <span lang="ja">四つ　</span> | <span lang="ja">よっつ</span>                                           |
| <span lang="ja">五つ　</span> | <span lang="ja">いつつ</span>                                           |
| <span lang="ja">六つ　</span> | <span lang="ja">むっつ</span>                                           |
| <span lang="ja">七つ　</span> | <span lang="ja">ななつ</span>                                           |
| <span lang="ja">八つ　</span> | <span lang="ja">やっつ</span>                                           |
| <span lang="ja">九つ　</span> | <span lang="ja">ここのつ</span>                                         |
| <span lang="ja">十　　</span> | <span lang="ja">とお</span> （这是一个例外，它不是量词） |

如你所见，这些词的发音极其烦人难学。不要试图全部记住，只需要知道它的存在，通过沉浸式接触来熟悉即可。同样的道理也适用于很多其他量词，如日（<span lang="ja">日</span>）、月（<span lang="ja">月</span>）、年（<span lang="ja">年</span>）、年龄（<span lang="ja">際</span>/<span lang="ja">才</span>）等。读法非常不规则。

关于年龄的话题，一个有趣的例外是，年龄有时直接用数字表示，不带量词：

<pre>
<span lang="ja">ボクはまだ</span><b>12</b><span lang="ja">ですよっ！</span>
我才12岁！
</pre>

如果你还记得之前的一课，量词的工作方式像副词，可以像副词一样放在句子的几乎任何位置。而且，就像副词一样，量词不能被格助词标记。虽然它们看起来像名词，但它们不是名词。

<pre>
<span lang="ja">二人、兄弟がいる</span>
<span lang="ja">兄弟が二人いる</span>
我有两个兄弟。
</pre>

这两个句子的意思相同（但后者更常见）。
