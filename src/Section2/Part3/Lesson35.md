# 形式名词与名词化：<span lang="ja">事</span>、<span lang="ja">物</span>、<span lang="ja">ところ</span> 和 <span lang="ja">の</span>

<span lang="ja">事</span>（<span lang="ja">こと</span>）和<span lang="ja">物</span>（<span lang="ja">もの</span>）都表示"东西"。<span lang="ja">事</span>通常用于无形的事物，如动作或状态，而<span lang="ja">物</span>通常用于有形的事物，如物体。在语法上，它们是名词，所以就像任何其他名词一样，它们可以被动词和形容词修饰。

<pre>
<span lang="ja">あ、いい</span><b><span lang="ja">こと</span></b><span lang="ja">考えた</span>
啊，我想到了个好<b>事情</b>。

<span lang="ja">美味しい</span><b><span lang="ja">もの</span></b><span lang="ja">が食べたい</span>
我想吃好吃的<b>东西</b>。
</pre>

<span lang="ja">所</span>（<span lang="ja">ところ</span>）是一个表示"地方"的词。它用在类似的模式中，包括关于时间中的瞬间或进展之类的无形"地方"，以及实体的某个方面。它经常缩写为<span lang="ja">とこ</span>。

<pre>
<span lang="ja">ちょうどいい</span><b><span lang="ja">ところ</span></b><span lang="ja">に来ました。</span>
你来得正<b>是时候</b>。（时刻 = 时间中的"位置"）

<span lang="ja">やれる</span><b><span lang="ja">とこ</span></b><span lang="ja">までやってみよう。</span>
让我们尽可能做下去吧。（字面：让我们尝试做到能做的地方为止）

<span lang="ja">こういう</span><b><span lang="ja">ところ</span></b><span lang="ja">が彼女のいい</span><b><span lang="ja">ところ</span></b><span lang="ja">だ。</span>
她的这种<b>地方</b>就是她的好<b>地方</b>（优点）。
</pre>

这三种形式名词都可以将整个短语变成名词。这让日语可以使用像事物一样的短语，而不需要使用嵌入从句。这基本上是在[第14课](../../Section1/Part1/Lesson14.md)中学过的关系从句的一种特殊简化情况。

这些形式名词是以下常见短语和表达的基础：

<pre>
<span lang="ja">無理をしなければ、死ぬ</span><b><span lang="ja">こと</span></b><span lang="ja">はないだろう。</span>
只要不过度勉强自己，就不会死吧。
</pre>

<span lang="ja">死ぬことはない</span>字面上类似于"不存在'会死'的事情/时刻"。

另一个是\<动词\><span lang="ja">たことがある</span>，字面上意思是"曾经做过\<动词\>的动作存在"，表示"曾经做过\<动词\>"：

<pre>
<span lang="ja">それはどこかで聞い</span><b><span lang="ja">たことがある</span></b><span lang="ja">。</span>
我<b>曾经</b>在哪里听<b>过</b>那个。
</pre>

<span lang="ja">こと</span>特别常用于名词化动词。在英语中，我们通过'-ing'结构做类似的事情："To play soccer"（踢足球）是一个动作，但"the act of play**ing** soccer"（踢足球的行为）是一个（抽象的）事物。

在日语中，我们也可以使用<span lang="ja">の</span>来达到类似的效果。在某些情况下，<span lang="ja">こと</span>和<span lang="ja">の</span>不能互相替换，但在名词化动作时，它们的用法在大多数情况下非常相似。

<pre>
<span lang="ja">私は歌を歌う</span><b><span lang="ja">の</span></b><span lang="ja">が好き</span>
我喜欢唱歌
</pre>

<span lang="ja">の</span>也可以用作形式名词，当我们在谈论什么可以从上下文中推断出来时，用它来替代更具体的词。这对形容词和动词都适用。

<pre>
<span lang="ja">赤い</span><b><span lang="ja">の</span></b><span lang="ja">はいいよ</span>
红色<b>的那个</b>可以哦。
</pre>
